# Reconsideration: Accessory Test Kitchen in Office Building

## Reconsideration #1 — 15 East 32nd Street, Manhattan

**Location:** 15 East 32nd Street, Manhattan | **Applicant:** Andrew Tuller | **Filing Type:** Alt Type II (ALT2) | **Building Code:** 1938 Building Code (Old Code) | **Occupancy Group:** Use Group 6 (UG 6) — Office | **Reviewer:** Max S. Lee, P.E., Acting Deputy Borough Commissioner | **Outcome:** APPROVED with conditions

**Issue:** Client (World Publications) wanted to install a Test Kitchen on the 11th floor of an office building exclusively for taking pictures for publications. The question was whether this required an amendment to the Certificate of Occupancy.

**Argument:** (1) As per the current CO, the 11th floor will remain occupied by offices. The proposed accessory non-residential kitchen will only be used by World Publications for photography. (2) No changes to existing zoning use because the accessory use is the same as existing office group 6 ("UG 6"). Under section C 27-217(b) of the NYC Building Code, a new CO is not required if the new use remains the same. No change in occupancy group classification because the proposed accessory kitchen remains commercial under the 1938 Building Code classification. This building is an "old code" building. (3) The number of persons on the entire floor will not exceed the ones allowed by current CO. (4) The kitchen will comply with all code requirements including range hood, exhaust system, fire suppression/ansul system filed separately.

**Conditions of Approval (handwritten by reviewer):** OK to accept test kitchen as stated. Notes on plan to indicate removal of kitchen at the end of tenant's lease. No sketch provided.

**Key Citations:** NYC Building Code Section C 27-217(b) — new CO not required if new use remains the same. 1938 Building Code occupancy classification.

## Reconsideration #2 — 16 East 34th Street, 14th Floor, Manhattan

**Location:** 16 East 34th Street, 14th Floor, Manhattan | **Applicant:** Andrew Tuller | **Filing Type:** Alt Type II (ALT2) | **Building Code:** 1938 Building Code (Old Code) | **Occupancy Group:** Use Group 6 (UG 6) — Office | **CO Number:** #81679 | **Date:** August 8, 2006 | **Reviewer:** Max S. Lee, P.E., Acting Deputy Borough Commissioner | **Outcome:** APPROVED with conditions

**Issue:** Client (Readers Digest magazine) wanted to install a Test Kitchen on the 14th floor of an office building exclusively for taking pictures. Same legal question — whether a CO amendment was required.

**Argument:** Identical legal framework as 15 E 32nd St. As per current CO #81679, the 14th floor remains offices. Accessory non-residential kitchen for Readers Digest employees only. No zoning use change (same UG 6). C 27-217(b) — no new CO required. Old code building (1938). Occupancy load unchanged. Kitchen will comply with all code requirements.

**Conditions of Approval (handwritten by reviewer):** OK to accept as stated. Applicant must provide proof of drawings. Provide C of O. Notes to provide removal of kitchen at end of tenant's lease term. Registered Architect stamp required (Andrew M. Tuller).

## Practical Impact

These two recons establish a consistent DOB pattern for accessory test kitchens in office buildings. The C 27-217(b) argument works reliably when: (1) the floor remains office use, (2) the kitchen is accessory to the primary office tenant, (3) occupancy load is unchanged, (4) kitchen code requirements (hood, exhaust, ansul) are filed separately. The DOB will condition approval on plan notes requiring kitchen removal at lease end. Both were approved by the same reviewer (Max S. Lee) confirming consistent application.

For old code buildings (1938 Building Code), the occupancy classification remains commercial regardless of the kitchen addition, so no reclassification is triggered.

When pricing these for clients, note that the ansul/fire suppression system is a separate filing — scope creep risk if not excluded in your proposal.
