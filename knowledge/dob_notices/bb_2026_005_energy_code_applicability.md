# BB 2026-005: When to Apply 2020 vs 2025 NYCECC Energy Code

## Overview

This bulletin supersedes BB 2020-002 and clarifies the applicable energy code requirements for New Building (NB) and Alteration (Alt) applications based on filing date cutoffs. The 2025 NYCECC creates new baseline energy compliance pathways while maintaining grandfather provisions for projects filed under the 2020 NYCECC.

## Regulatory Authority

- **1 RCNY 5000-01** (New York City Energy Conservation Code)
- Local Law 97 (Climate Mobilization Act)
- LL 159 (Energy Code Updates)
- Relationship to LL 153 and LL 97 compliance timelines

## Applicability Effective Dates

### 2020 NYCECC (NYCECC-2020)
- **Applies to:** NB and Alt applications filed on or before [VERIFY: March 30, 2026]
- Baseline energy compliance via prescriptive path or performance path
- TR8 progress inspections follow 2020 code requirements
- Grandfathered provisions remain in effect for ongoing projects

### 2025 NYCECC (NYCECC-2025)
- **Applies to:** NB and Alt applications filed on or after [VERIFY: March 31, 2026]
- Enhanced energy efficiency baseline requirements
- Updated calculation methodologies for building energy code compliance
- New renewable energy and on-site generation requirements
- Tighter envelope performance standards

## Application Types and Code Assignment

### New Building (NB) Applications

**2020 NYCECC Applies If:**
- Objection/objection review filed before cutoff date
- Application meets all 2020 NYCECC prescriptive or performance path requirements
- No amended scope that extends project timeline beyond grace period

**2025 NYCECC Applies If:**
- First filing date is on or after cutoff date
- Voluntary compliance election (rare; must be explicit in scope)

**Critical Note:** NB applications filed under 2020 code that undergo substantial modification or phased amendments may trigger 2025 code requirements for amended scope portions [VERIFY scope rules].

## Code Compliance Paths

### 2020 NYCECC Compliance Methods

**Prescriptive Path (2020):**
- Envelope U-value requirements
- HVAC efficiency minimum standards (ASHRAE 90.1-2019 reference)
- Lighting power density limits
- All requirements per NYCECC-2020 Table 501, etc.

**Performance Path (2020):**
- Baseline building performance (EUI-based)
- Design building performance modeling
- Compliance via whole-building energy simulation

### 2025 NYCECC Compliance Methods (Enhanced Requirements)

**Prescriptive Path (2025):**
- Tighter envelope U-values (approximately 15-20% reduction from 2020 baseline)
- Increased renewable energy requirement (minimum percentage of building load)
- On-site generation capacity threshold [VERIFY: kW minimum]
- HVAC efficiency stepped to ASHRAE 90.1-2022 reference standards

**Performance Path (2025):**
- Lower baseline EUI allowance
- Mandatory renewable energy component (cannot be offset by efficiency alone)
- Grid harmonization credit pathway [VERIFY: definition]
