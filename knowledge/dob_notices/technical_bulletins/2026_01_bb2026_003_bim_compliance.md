# Buildings Bulletin 2026-003: Voluntary Use of BIM Compliance Check Software

**Issued January 21, 2026** — Guidance for the voluntary use of BIM compliance check software to analyze digital 3D models for compliance with select NYC Code provisions prior to DOB submission.

## Background

From the 2021 Hack the Building Code Innovation Challenge, DOB recognized software that checks BIM plans for Code compliance. This bulletin formalizes the voluntary use of such software at the pre-filing stage.

## What It Is

BIM compliance check software analyzes digital 3D models for compliance with select NYC Code provisions before plan submission. Produced by independent organizations, not DOB.

## How It Can Be Used

- At the discretion of the Registered Design Professional (RDP)
- Used at the **pre-filing stage** only
- Checks compliance with applicable Codes, rules, and regulations
- Helps catch issues before DOB submission

## Critical Limitations

1. **DOB does NOT evaluate** the software — reliability and accuracy must be assessed by the user
2. **NOT a replacement** for current DOB submission requirements — digital models and reports from BIM software are not qualified replacements or supplements
3. All submissions must still comply with:
   - NYC Construction Codes
   - Minimum Requirements for Review of Design Drawings
   - Drawing Standards for Plan/Work Application
   - Project Guidelines for Design Professionals

## Related Code Sections

- BC 107 (Construction documents)
- AC 28-104 (Submission requirements)

## Practical Notes

- This is informational only — it doesn't change any filing requirements
- Could be useful for architects doing self-cert work to catch issues pre-submission
- DOB is signaling openness to technology in the review process
- Does not currently speed up or change the plan examination process
