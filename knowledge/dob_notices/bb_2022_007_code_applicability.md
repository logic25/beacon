# BB 2022-007: When to Apply 2014 vs 2022 Construction Codes

## CRITICAL RULE FOR EXPEDITERS
**The filing date determines which code applies — this is the single most important decision point in expediting work.**

---

## EFFECTIVE DATE OF 2022 CODES
- **2022 NYC Construction Codes became effective: November 7, 2022**
- Authority: Local Law 126 of 2021
- These codes apply to all buildings and structures except as otherwise provided in this section

---

## PRIMARY CODE DETERMINATION RULE

### Applications Filed ON OR AFTER November 7, 2022
- **2022 NYC Construction Codes apply**
- This applies regardless of when construction work begins
- Once filed with 2022 code, the entire application is subject to 2022 code unless scope changes

### Applications Filed BEFORE November 7, 2022
- **2014 NYC Construction Codes apply**
- This rule applies even if the work has not yet started or been permitted
- This provides grandfather protection for pre-effective date filings
- Applicant cannot voluntarily upgrade to 2022 code to access new provisions

---

## CRITICAL EXCEPTION: CHAPTER 33 (SAFEGUARDS DURING CONSTRUCTION/DEMOLITION)

### BC Chapter 33 Applies Regardless of Filing Date
- 2022 BC Chapter 33 (Safeguards During Construction and Demolition) applies to ALL **active construction sites**
- Chapter 33 applies regardless of whether the underlying application was filed before or after November 7, 2022
- This includes:
  - Shoring requirements
  - Protection of adjoining properties
  - Demolition safety requirements
  - Temporary protective systems
  - Site safety plans

**EXPEDITER ALERT:** This creates a hybrid situation where an old project filed in 2022 may be under 2014 code for design/construction requirements but 2022 BC Chapter 33 for active site operations.

---

## DEMOLITION PROJECTS

### Full Demolition
- If DM (Demolition) application filed **ON OR AFTER November 7, 2022** → 2022 code applies
- If DM application filed **BEFORE November 7, 2022** → 2014 code applies
- 2022 BC Chapter 33 still applies to active demolition sites

### Partial Demolition
- Same rule applies as full demolition — filing date determines code
- If partial demo is part of an NB or ALT application, the application filing date determines code for that entire project

---

## SIGN-OFF APPLICATIONS (PAA)

### For Pre-November 7, 2022 Applications
- PAAs (Progress or Application Amendments / sign-off applications) remain under 2014 code
- This protects applicants from unexpected code upgrades
- **EXCEPTION: If scope change triggers a new filing, new filing is under current code (2022 if filed after 11/7/2022)**

### Scope Changes
- Minor amendments/amendments that don't change fundamental scope: stay under original code
- [VERIFY] Material scope changes that essentially create new work may trigger 2022 code application
- Consult with design team on whether modification constitutes new filing vs. amendment

---

## PARTIAL EFFECTIVE DATES AND TRANSITION PROVISIONS

### Some Provisions Have Delayed Effective Dates
- Not all 2022 BC provisions effective November 7, 2022
- [VERIFY] Check specific sections — some provisions have staggered effective dates
- Example provisions to verify:
  - Energy code adoption date
  - Accessibility standards updates
  - Specific technical requirements in various chapters

**ACTION ITEM FOR EXPEDITERS:** Always check specific code section citations in plans/specs against BB 2022-007 Appendix [VERIFY - verify appendix exists with delayed dates]

## PRACTICAL EXPEDITING SCENARIOS

### Scenario 1: Application Filed October 15, 2022
- **Code:** 2014 NYC Construction Codes (entire application)
- **Chapter 33:** 2022 BC Chapter 33 applies once construction begins
- **Impact:** Design must comply with 2014 code; site safety must comply with 2022 Chapter 33

### Scenario 2: ALT Application Filed November 10, 2022
- **Code:** 2022 NYC Construction Codes (entire application)
- **Chapter 33:** 2022 BC Chapter 33 applies (same code)
- **Impact:** All design and site requirements follow 2022 code

### Scenario 3: DM Application Filed in 2022; Demolition Begins in 2023
- **Code:** Depends on filing date, not start date
- If filed before 11/7/2022 → 2014 code governs demolition approach
- If filed after 11/7/2022 → 2022 code governs demolition approach
- **Chapter 33:** 2022 BC Chapter 33 applies to active demolition site

### Scenario 4: NB Filed September 2022; PAA Sign-Off Filed January 2025
- **Code:** 2014 code applies to NB and PAA (same filing "family")
- **Exception:** If PAA is major scope change, consult DOB on whether new filing required

## CRITICAL REMINDERS FOR EXPEDITERS

1. **Always confirm filing date first** — this determines code applicability for entire project
2. **Chapter 33 is always 2022** — even if rest of project is under 2014 code
3. **Filing date is in the system** — DOB NOW shows filing date; verify in case of amendments
4. **Scope changes may require new filing** — consult DOB if substantial modifications proposed
5. **Staggered effective dates exist** — for certain provisions, verify specific sections
6. **Applicant cannot choose code** — they follow the code in effect on filing date

---

## COMMON EXPEDITER ERRORS TO AVOID

❌ Assuming project filed Nov 2022 must use 2022 code (check exact filing date)
❌ Thinking Chapter 33 requirements depend on application filing date (they don't)
❌ Assuming PAA must comply with 2022 code if PAA filed after 11/7/2022 (it doesn't)
❌ Overlooking staggered effective dates for specific provisions
❌ Not flagging code applicability issue early in design/submission process
