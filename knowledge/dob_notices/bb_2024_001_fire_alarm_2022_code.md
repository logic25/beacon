# BB 2024-001: Fire Alarm System Work Under 2022 Construction Codes

## Overview

This bulletin supersedes BB 2015-025 and provides updated guidance on fire alarm system compliance requirements under the 2022 Construction Codes. It clarifies which types of work involving existing fire alarm systems designed under the 2014 NYC Building Code (or prior) triggers compliance upgrade to the 2022 standards and which work can remain under the original design code.

## Applicability Framework

### Alteration Definition for Fire Alarm Systems

Per AC 28-101.4.3, an **Alteration** includes:
- Addition of fire alarm equipment or circuits
- Replacement of fire alarm devices or panels
- Modification of fire alarm logic or programming
- Extension of fire alarm coverage to previously unprotected areas
- Upgrade of signal transmission method (hardwired, wireless, cellular)

**Key Distinction:** Repair vs. Alteration
- **Repair** (maintenance): In-kind replacement of identical device at same location; no upgrade to new code
- **Alteration**: Any design change, expanded scope, modernization, or system upgrade; triggers 2022 code

## Types of Fire Alarm Work and Code Applicability

### Category 1: Repair/Maintenance (No Code Upgrade Required)

**Work Scope:**
- Replacement of initiating devices (smoke detectors, heat detectors, manual pull stations) with same or equivalent model
- In-kind replacement of notification devices (speakers, strobes, bells) at same location
- Repair of broken wiring within same route and gauge
- Battery replacement in fire alarm panels
- Software security patches to existing panels (no logic change)
- Scheduled maintenance and testing

**Code Applicable:** Original design code (2014 BC or prior)

**Filing Requirement:** Check with DOB — may qualify as Maintenance/Repair exempt from filing, or may require Alt with limited scope statement [VERIFY: current DOB position on repair-only scope]

**Documentation:** Fire alarm maintenance plan; service records

---

### Category 2: Limited Expansion/Alteration (Triggers 2022 Code — Partial Scope)

**Work Scope:**
- Addition of fire alarm zone to cover new or previously unprotected tenant space
- Expansion of detection coverage within building (e.g., adding smoke detectors to additional floors)
- Integration of separate fire alarm systems into unified system
- Upgrading signal transmission method (from hardwired to networked; from dial-up to IP)
- Panel firmware upgrade with logic or programming changes
- Addition of accessible manual pull stations (ADA compliance upgrade)
- Addition of visual notification devices (strobes) for ADA compliance

**Code Applicable:** 2022 BC §907 and §915 (full current code requirements)

**Filing Requirement:** Full Alt Alt (or Full Alt if NB building getting fire alarm for first time)

**Scope Trigger:** Once any Category 2 work is performed, entire fire alarm system upgrade to 2022 code standard [VERIFY: whether entire building or just modified zones]

**Design Requirements (2022 Code):**
- NFPA 72 (2019 edition) compliance [VERIFY: which version referenced by 2022 BC]
- Manual pull stations on every floor, spacing per BC 907.2.11.1
- Visual notification devices per BC 907.8.4 (ADA)
- Battery backup and power calculations per NFPA 72, Chapters 6 and 10
- Fire alarm riser diagram with single-line schematic
- System layout plan at scale
- Device schedule with models and specifications
- Integration specifications if connecting to existing systems

---

### Category 3: System Replacement or Significant Modernization (Mandatory 2022 Code)

**Work Scope:**
- Complete replacement of fire alarm control panel
- Installation of new fire alarm system in building with no prior system
- Major upgrade of detection methodology (analog addressable replacing conventional, etc.)
- Conversion to networked/distributed architecture
- Replacement of more than [VERIFY: 50%?] of initiating or notification devices
- Retrofit for Modern Code-Compliant Features (e.g., converting to accessible manual pull stations throughout)

**Code Applicable:** 2022 BC §907, §915, and full Chapter 9 (Fire and Life Safety Systems)

**Filing Requirement:** Full Alt Alt with comprehensive fire protection scope

**Design Requirement Level:** Maximum — entire system re-designed to 2022 standard

---

**Work Scope:**
- Emergency repair of failed fire alarm system
- Temporary fire alarm system pending permanent installation
- Emergency upgrade due to DOB violation or Fire Department order

## Existing Fire Alarm Systems Designed Under 2014 BC or Prior

### When Old System Can Remain (No Upgrade Required)

An existing fire alarm system designed and installed under the 2014 NYC Building Code (or earlier) **may continue to operate without code upgrade** if:

1. **No alterations are made** to the system (Category 1 repair/maintenance only)
2. **No changes to protected premises** (no new occupancies, no expanded coverage area)
3. **No changes to trigger conditions** (no modification of detection logic or alarm thresholds)
4. **System continues to function as designed** (no degradation of performance)
5. **Maintenance records are current** (annual inspection and testing)

**Exception:** [VERIFY: whether systems over certain age (20+ years?) must be upgraded regardless of work scope]

### When Old System Must Upgrade to 2022 Code

An existing fire alarm system designed under 2014 BC (or prior) **must be upgraded to 2022 code** if any of the following occur:

1. **Any alteration work** per Categories 2 or 3 above
2. **Building alteration** that changes occupancy, use, or protected area (trigger: AC 28-101.4.3)
3. **Change of occupancy** classification requiring new or expanded fire protection
4. **DOB violation** citation requiring fire alarm upgrade
5. **FDNY order** (via N-5 notice or other Fire Department directive)
6. **Building undergoing Major Alteration** per AC 28-101.4.3 (approximately 25%+ of building envelope or systems) — fire alarm typically included in scope [VERIFY: specific Major Alt trigger]

## Alteration Trigger Analysis for Fire Alarm Compliance Upgrade

### Scenario 1: Tenant Alteration in Occupied Building

**Fact Pattern:** Existing office building with 2014 BC fire alarm system. Tenant Alt in floors 5-7 to reconfigure office layout and add new conference room.

**Analysis:**
- Tenant Alt scope limited to interior partition changes, no HVAC or electrical changes
- Existing fire alarm system serves entire building
- Altered area remains same occupancy (office)

**Conclusion:** Fire alarm system does not require upgrade to 2022 code.
- Existing system adequate for unchanged occupancy
- No expansion of protected area
- No modification to fire alarm devices or logic
- Filing: Tenant Alt scope statement excludes fire alarm [VERIFY: explicit exclusion language recommended]

---

### Scenario 2: Mixed-Use Building Adding New Residential Use

**Fact Pattern:** Building with existing 2014 BC fire alarm system (designed for commercial). Owner adds new residential floor via floor addition (Major Alteration triggered).

**Analysis:**
- Change of occupancy: addition of residential use group requires enhanced fire protection
- Residential occupancy (Group R) typically has different fire alarm requirements than commercial (Group B)
- Expanded protected area and new detection zones required

**Conclusion:** Fire alarm system MUST upgrade to 2022 code for entire building (or at minimum, new residential addition and interconnected zones).
- Full Alt Alt required
- 2022 BC §907 design standards apply
- New detection scheme per occupancy classifications in 2022 code
- Manual pull stations, visual notification devices per ADA/2022 requirements

---

### Scenario 3: Replacing Fire Alarm Panel Due to Age

**Fact Pattern:** 2014 BC fire alarm system reaches end of manufacturer support (panel is 18 years old). Owner elects to replace control panel with modern addressable panel; detection devices (smoke detectors, heat detectors) unchanged.

**Analysis:**
- Panel replacement is alteration work (Category 3)
- Even though detection devices unchanged, new panel represents system modernization
- New panel will have different programming, addressing, and logic capabilities

**Conclusion:** Full system upgrade to 2022 code required.
- New panel must comply with 2022 NFPA 72 and BC §907
- Existing detection device locations reviewed for compliance with 2022 spacing and coverage (BC 907.2)
- Alt Alt scope includes panel replacement + any remedial detection upgrades needed for 2022 compliance
- Riser diagram and device schedule must be re-drawn per 2022 standards

---

### Scenario 4: HVAC Major Alteration — Fire Alarm Not Directly Affected

**Fact Pattern:** Building undergoing Major Alteration (25%+ of HVAC system replacement). Existing 2014 BC fire alarm system is unmodified; no work on fire alarm devices or circuits.

**Analysis:**
- Major Alteration definition (AC 28-101.4.3) triggers building code upgrade for affected systems
- Fire alarm is separate building system, not part of HVAC scope
- Question: Does Major Alt trigger fire alarm upgrade even if fire alarm scope is zero?

**Conclusion:** [VERIFY DOB position — typically fire alarm is included in Major Alt scope even if not directly modified]
- Recommend: File Major Alt including fire alarm system upgrade to 2022 code

- Conservative approach: Include fire alarm in any Major Alt filing

## NYC Electrical Code Integration

### Fire Alarm Power Requirements (2022 Standards)

**Primary Power:**
- Fire alarm control panel must have dedicated circuit per NEC Article 700 (Emergency Systems)
- Minimum #12 AWG copper wire (2022 updates may specify larger)
- Disconnect switch within 6 feet of panel per BC 907.6
- Voltage drop calculation per NFPA 72 Chapter 6

**Backup Power (Battery):**
- Minimum 24-hour standby + 5-minute alarm capability
- Battery supervision alarm per NFPA 72
- Heat sensor or thermal cutoff per battery safety standards

**NYC Electrical Code Compliance:**
- All electrical work on fire alarm circuits must comply with NYC Electrical Code
- Fire alarm wiring separation from low-voltage data circuits per code
- Conduit sizing and protection requirements

**Interaction with Electrical Alt:** If fire alarm upgrade includes new wiring or circuit modifications, coordinate with electrical permits/inspections [VERIFY: combined filing or separate electrical work order]

## Documentation and Filing Requirements

### 2022 Code Compliance Submission (Category 2 or 3 Work)

**Sealed Drawings and Specifications (PE/fire protection professional):**
1. Fire Alarm System Riser Diagram
   - Single-line schematic showing all circuits and zones
   - Panel location, battery location
   - Manual pull station locations
   - Notification device locations (speakers, strobes)
   - Integration with other building systems (sprinkler, smoke control, etc.)

2. Fire Alarm Device Layout Plan(s)
   - Floor plans at scale showing initiating devices (detectors, pull stations)
   - Notification devices (speakers, visual alarms)
   - Wiring routes and cable trays (if exposed)
   - Device spacing calculations demonstrating compliance with BC 907.2.11

3. Equipment Schedule
   - Control panel: model, addressable/conventional, capacity
   - Detectors: type, model, spacing, coverage area
   - Manual pull stations: model, accessibility features
   - Notification devices: type, model, sound level (dB calculation for ADA)
   - Power supplies, batteries: amp-hour capacity, runtime calculation
   - Signal transmission: hardwired, wireless, networked, cellular

4. Specifications Section
   - NFPA 72 edition referenced (typically 2019 per 2022 BC)
   - Detector sensitivity thresholds
   - Manual pull station spacing: not to exceed 200 feet travel distance (BC 907.2.11.1)
   - Visual notification device requirements for ADA compliance (BC 907.8.4)
   - Testing and inspection requirements
   - Acceptance testing procedure (NFPA 72 Chapter 14) [VERIFY: DOB or FDNY acceptance role]

5. Integration Documentation (if applicable)
   - Connection to existing systems (sprinkler, smoke control, damper circuits)
   - Sequence of operations (e.g., activation of HVAC shutdown upon alarm)
   - Signal transmission method security (encryption for networked systems) [VERIFY: requirement]

## Code Comparison: 2014 BC vs. 2022 BC Fire Alarm Requirements

| **Requirement** | **2014 BC** | **2022 BC** | **Impact** |
|---|---|---|---|
| Manual pull station spacing | 250 ft travel distance | 200 ft travel distance | Increased density of pull stations required |
| Strobe light candlepower (ADA) | Per NFPA 72 (version as of 2014) | Enhanced per NFPA 72 (2019); 75 cd minimum | Brighter, more visible strobes needed |
| Accessible pull station height | 42"-48" AFF | 40"-56" AFF | Expanded accessible range [VERIFY] |
| Smart/networked alarms | Limited guidance | Full pathway in NFPA 72 2019 | IP-based systems now fully supported |
| Two-way emergency communication | Optional in many cases | Expanded requirement per BC 907.6.5 [VERIFY] | Voice alarm system and communication network required in certain occupancies |
| Smoke detection in HVAC ducts | Specified | Enhanced per BC 915 | Additional ductwork detection may be required |
| Integration with smoke control | Permitted but not mandated | More prescriptive per BC 907.12.3 [VERIFY] | Automatic smoke control activation upon alarm |

## Common Mistakes and Expediting Pitfalls

### Mistake 1: Assuming Repair Triggers No Upgrade

**Error:** Building owner replaces 50% of smoke detectors under "maintenance contract." Expeditor files no alteration.

**Risk:** DOB discovers work during inspection; issues violation for work without permit. System may not comply with 2022 standards.

**Solution:** Clarify with owner and fire alarm contractor whether work is in-kind repair (Category 1) or system modernization (Category 2/3). If any design change, file Alt Alt.

---

### Mistake 2: Excluding Fire Alarm from Major Alteration

**Error:** Building Major Alt scope includes HVAC, electrical, and structure; fire alarm explicitly excluded to avoid scope creep.

**Risk:** DOB may flag fire alarm system as not upgraded despite Major Alt, creating compliance gap. Future inspector notes building Major Altered but fire protection outdated.

**Solution:** Include fire alarm in Major Alt scope with statement of work (upgrade to 2022 code, or assessment showing existing system adequate). Document justification if truly excluding.

---

### Mistake 3: Filing Under Old Code After 2022 BC Effective Date

**Error:** Expeditor files Alt Alt in 2024 on old 2014 BC template; fire alarm scope included but design per 2014 standards.

**Risk:** Plan examiner rejects for not conforming to 2022 BC. Resubmission required with 2022 code compliance.

**Solution:** [VERIFY: DOB's code applicability rule for fire alarm specifically]
- Generally: Applications filed after 2022 BC effective date must comply with 2022 code
- Confirm filing date determines applicable code
- Request extension/clarification from DOB if project has pre-2022 filing dates

---

### Mistake 4: Mixing Existing and New Fire Alarm Standards

**Error:** Reusing 2014 BC-compliant manual pull stations in new 2022 code-designed system.

**Risk:** Pull stations may not meet 2022 spacing (200 ft, not 250 ft) or accessibility requirements (40-56" height). System incomplete or non-compliant.

**Solution:** Schedule entire building for pull station audit when upgrading to 2022. Budget for new, ADA-compliant units at required spacing.

## FAQs for Expeditors

**Q: If a building has a 2014 BC fire alarm system and we're doing a tenant alteration, do we need to upgrade the fire alarm?**
A: Only if the tenant alteration itself involves fire alarm work (expanding zones, replacing devices, changing logic). If the tenant's work does not touch fire alarm, the system may remain unchanged. However, if the tenant's use changes (e.g., office to residential), the fire alarm may need to be re-designed for new occupancy. Verify with DOB Plan Examiner.

**Q: We're replacing a fire alarm panel in a 2014 BC system. Does the entire building fire alarm need to be upgraded to 2022 code?**
A: [VERIFY DOB guidance — likely yes, panel replacement triggers system-wide upgrade to 2022 standards]

**Q: Manual pull stations in our 2014 BC building are spaced at 250 feet apart. If we upgrade to 2022 code, do we need to add more?**
A: Yes. 2022 BC requires 200-foot spacing. Budget for additional pull stations at locations filling gaps between existing stations.

**Q: Can we use wireless fire alarm devices to avoid rewiring existing buildings?**
A: Yes, wireless systems are compliant with 2022 NFPA 72 and BC §907, if designed per NFPA 72 Chapter 23. However, wireless system approval/acceptance may require additional testing and FDNY coordination [VERIFY].
