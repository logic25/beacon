## Most Referenced BBs for Expediters

These are the highest-priority BBs that impact most filings and should be consulted first:

### BB 2022-007 — Code Applicability (CRITICAL)
- **Date:** May 10, 2022
- **Status:** ACTIVE
- **Topic:** When to apply 2014 vs 2022 Construction Codes
- **Impact:** Affects EVERY filing. Determines which code edition applies to work performed, permits issued, filing dates, etc.
- **Code Sections:** AC 28-101.4, BC 3310, LL126/2021
- **Reason to Check:** Most filings require verification of code applicability; this is the authority on the 2014/2022 transition.

### BB 2026-005 — Energy Code Applicability
- **Date:** Feb 17, 2026
- **Status:** ACTIVE
- **Topic:** When to apply 2020 vs 2025 NYCECC
- **Impact:** Determines which energy code applies; supersedes BB 2020-002
- **Code Sections:** 1 RCNY 5000-01
- **Reason to Check:** Energy code compliance is now driven by this bulletin; critical for LL97, LL128, energy-related filings.

### BB 2025-005 — Professional Certification Program
- **Date:** Jun 3, 2025
- **Status:** ACTIVE
- **Topic:** Professional Certification Program procedures and requirements
- **Impact:** Governs pro-cert filing procedures, scope, and acceptance criteria; supersedes BB 2016-010 and OPPN #1/04
- **Code Sections:** AC 28-104.2, RCNY 21-02, MC/PC/BC/FGC/FC 105-107
- **Reason to Check:** Any filing with a professional certificate must follow procedures in this bulletin.

### BB 2025-011 (Rev1) — Structural Filing and Permit Requirements
- **Date:** Sep 30, 2025 (Revised Oct 28, 2025)
- **Status:** ACTIVE
- **Topic:** New Building and Alternate structural filing and permit requirements
- **Impact:** Governs structural scope for permits under both 2014 and 2022 codes
- **Code Sections:** AC §28-101.4.5, §28-104, §28-105, BC §107.3, §1603.1
- **Reason to Check:** Any structural work filing must reference this bulletin for scope determination.

### BB 2025-003 — TR8 Energy Progress Inspections
- **Date:** Apr 3, 2025
- **Status:** ACTIVE
- **Topic:** TR8 energy progress inspection schedule and format
- **Impact:** Governs energy inspection procedures and filing requirements
- **Code Sections:** 1 RCNY 5000-01, 1 RCNY 101-07, 2020 NYCECC
- **Reason to Check:** Any energy work progress inspection must follow this bulletin.

### BB 2025-002 — Amended CO for City of Yes
- **Date:** Apr 1, 2025
- **Status:** ACTIVE
- **Topic:** Amended Certificate of Occupancy for small establishments under City of Yes zoning
- **Impact:** Governs CO amendments for City of Yes zoning compliance
- **Code Sections:** AC 28-118.3, ZR 11-24
- **Reason to Check:** Any CO amendment for City of Yes zoning changes must follow this bulletin.

### BB 2024-001 — Fire Alarm Under 2022 Code
- **Date:** Feb 9, 2024
- **Status:** ACTIVE
- **Topic:** Fire alarm work under 2022 code applicability
- **Impact:** Supersedes BB 2015-025; governs fire alarm filing scope and procedures
- **Code Sections:** AC 28-101.4.3, BC 901.9/907/915, Elec Code
- **Reason to Check:** Any fire alarm work filing must reference current procedures in this bulletin.

## How to Use This Index

### For Filings
1. Identify the work category (structural, electrical, mechanical, etc.)
2. Check the relevant BBs in the "Most Referenced" section
3. Verify the code applicability using **BB 2022-007** (2014 vs 2022) and **BB 2026-005** (energy code)
4. For professional certification work, consult **BB 2025-005**
5. For any superseded document, follow the NEWER BB (never use rescinded or superseded documents)

### For Compliance Questions
1. Search this index by subject or code section
2. Check if the BB is marked ACTIVE (use it) or SUPERSEDED/RESCINDED (find the replacement)
3. Follow the "Supersession Chain" to find the current document if a BB has been replaced
4. Cross-reference code sections to ensure full compliance

### For Knowledge Base (RAG) Queries
- When a user asks about a specific BB, this index provides its current status
- When a user asks about a topic covered by multiple BBs, check the "Supersession Chain" to identify which BB is current
- Use the "Most Referenced" section to prioritize which BBs to pull into context for common questions
- Rescinded documents should NOT be cited; always direct to active replacements

### For Document Verification
- If you encounter a reference to a BB from before 2022, check if it has been superseded (use "Supersession Chain")
- If a BB is marked RESCINDED, verify the replacement document
- BBs from 2024-2026 should be assumed ACTIVE unless marked otherwise

## Status Definitions

| Status | Meaning | Action |
|--------|---------|--------|
| **ACTIVE** | Current and applicable. Follow this bulletin. | Use for compliance and filings. |
| **SUPERSEDED** | Replaced by a newer BB. Do not use. | Reference the newer BB listed in "Supersedes" column. |
| **RESCINDED** | Formally cancelled. Do not use. | Function may have moved to website or other guidance. |
| **ACTIVE (Rescission)** | This BB is active AND contains rescission of older documents. | Use this BB AND note which old docs it rescinded. |
