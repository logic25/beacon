# Second Review of Plan Examination Objections

## Determination Request Fees (Effective January 28, 2018)

Per 1 RCNY 101-03:

| Request Type | Fee |
|---|---|
| CCD1 or ZRD1 request for Determination | $1,000 |
| Appeal of CCD1 or ZRD1 Determination | $2,500 |

**CCD1/ZRD1 Determination covers:**
- Pre-Determination (pre-filed job) Request
- Request for variation of a Code requirement or MDL section 277.16
- Appeal of an affirmation of an objection after a second plan examination review

**Fee Exemptions:** Properties exempt per AC 28-112.1 and 1 RCNY 101-03

## Second Plan Examination Review Process (AI1)

**Before submitting a Determination request**, a second plan examination review is required. This review is covered by the original filing fee — no additional fees.

### How to Request:
1. Complete an **AI1 form** listing the objection and reasons for removal
2. Include all supporting documentation in **one flattened PDF**
3. Submit to the appropriate borough email address

### Borough Email Addresses (DOB NOW: Build filings)

Use the email specific to the **assigned Plan Examiner's location**:

| Borough | Email |
|---|---|
| Bronx | BXObjectionReview@buildings.nyc.gov |
| Brooklyn | BKObjectionReview@buildings.nyc.gov |
| Manhattan | MNObjectionReview@buildings.nyc.gov |
| Queens | QNObjectionReview@buildings.nyc.gov |
| Staten Island | SIObjectionReview@buildings.nyc.gov |
| NYC Development HUB | DHubObjectionReview@buildings.nyc.gov |
| HUB Full Service | HubFSObjectionReview@buildings.nyc.gov |

### Possible Outcomes:
1. **Resolved** — Objection removed
2. **Rejected** — Objection upheld
3. **Meeting required** — Borough office schedules appointment; applicant must be present with disapproved plans and all relevant information

### If Second Review is Rejected:
- **Option A:** Revise the plans to address the objection
- **Option B:** Submit a Determination request to appeal the affirmation

## Determination Request Requirements

- **Zoning:** File ZRD1 (Zoning Resolution Determination Form)
- **Construction Code:** File CCD1 (Construction Code Determination Form)
- Must include copy of **invoice from Borough Office** where fee was paid
- Forms must be **signed and sealed by a professional**
- Limited to **one determination per form**
- Must cite **all related Code or Zoning sections**
- Attachments must be **11" x 17" paper or smaller**

## Practical Tips

- Always exhaust the free AI1 second review before paying $1,000 for a determination
- Make the AI1 thorough — cite specific code sections, include annotated drawings
- Submit as one flattened PDF, not multiple attachments
- If the examiner is at the Development Hub, use the DHubObjectionReview email, not the borough email
- Keep track of which borough office the plan examiner is assigned to — it may not match the building's borough
