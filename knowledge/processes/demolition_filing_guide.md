# NYC Demolition Filings - Complete Process Guide

## Overview

This guide provides expediting professionals with step-by-step procedures for navigating NYC Department of Buildings (DOB) demolition filings. It covers both full and partial demolitions, from pre-application planning through final permit issuance.

---

## Part 1: Demolition Types and Applicability

### Full Demolition

**Definition & When It Applies:**
- Complete removal of an entire building structure
- Demolition of all buildings on a lot
- Applicable when the entire structure is unsafe, obsolete, or subject to redevelopment
- Typically filed as a standalone application (Permit Type DM)

**Key Characteristics:**
- Single demolition permit covers the entire structure
- Must address all utilities servicing the building
- Site becomes vacant lot post-demolition
- Requires special attention to utility discontinuation orders

**Pre-Filing Considerations:**
- Verify building is not designated as historic (NYC Landmarks Preservation Commission)
- Check if partial preservation is required by any constraints
- Confirm zoning allows demolition and future use

### Partial Demolition

**Definition & When It Applies:**
- Removal of one or more buildings on a lot with other structures remaining
- Interior gutting while maintaining the facade (may qualify as Alteration, not Demo)
- Demolition of annexes, extensions, or secondary structures
- Removal of specific floors or sections within a building

**Key Characteristics:**
- Separate permits may be required for each building being demolished
- Utility work more complex if some buildings remain served
- Site Safety Coordinator requirements differ from full demo
- Adjacent building protections are critical

**Pre-Filing Considerations:**
- Determine which structures are actually being demolished
- Identify which utilities serve each remaining structure
- Plan for ongoing operations of remaining buildings
- Budget for NB (Neighboring Building) protections

**Common Partial Demo Scenarios:**
- Demolition of carriage house with main building remaining
- Removal of non-load-bearing annexes from industrial buildings
- Side-by-side demolition on a lot with multiple structures
- Removal of mechanical penthouse only

## Part 2: Filing System and Forms Requirements

### DOB NOW vs. BIS: Which System to Use

**DOB NOW (Modern System - Preferred):**
- Used for most demolition filings filed after 2015
- Digital-first platform with better tracking
- Most permits issued faster through DOB NOW
- Allows for online payment and communication
- Mobile inspection routing integrated

**When to Use DOB NOW:**
- Standard full demolitions
- Standard partial demolitions
- Routine asbestos abatement demolitions
- Emergency demolitions (special routing available)

**BIS (Legacy System - Limited Use):**
- [VERIFY] May still be required for certain historical buildings
- Used for specific Landmarks Preservation Commission coordination cases
- Occasionally required for unique historic preservation conditions
- Less common in current practice; confirm with DOB before filing

**How to Determine Correct System:**
1. Check DOB's website for applicant guidelines
2. Contact DOB's Expediting Services division if uncertain
3. When in doubt, DOB NOW is the default for new applications
4. File with applicant's standard system if ongoing relationship exists

### Demolition Permit Types and Forms

**Standard Demolition Permit (DM)**

Primary form for all demolitions:
- **Form Required:** DM-1 (Application for Permit - Demolition)
- **Description:** General demolition permit covering structural removal
- **Filed Through:** DOB NOW or BIS
- **Applicant:** Licensed General Contractor holding Demolition Classification
- **Timeline:** 15-45 days from complete filing

**Components of Complete DM Filing:**

1. **DM-1 Application Form**
   - Building information (address, block/lot, 4-digit ID number)
   - Applicant information (contractor license number)
   - Owner information and authorization
   - Superintendent of Schools notification certification
   - Estimated demolition cost
   - Target demolition date

2. **Support Documents**
   - Proof of site control (deed, lease, or owner authorization letter)
   - License documentation (General Contractor with Demolition Classification)
   - Insurance documentation (see Part 8)
   - Asbestos certification (ACP-5 form or waiver)
   - Utility disconnection notification proof (see Part 3)

3. **Site Safety Plans**
   - Designation of Site Safety Manager (form SSM-1)
   - Site Safety Coordinator information (form SSC-1) if applicable
   - Site Safety Plan (CEQR required, see Part 4)

4. **Neighboring Building Documentation** (if applicable)
   - NB-1 Form (Affidavit of Proof of Service)
   - NB-2 Form (Survey of Neighboring Buildings - for adjacencies within 25 feet)
   - Shoring and protection plans
   - Pre-demolition condition photographs

### Additional Regulatory Filings (Often Coordinated with DM)

**ACP-5 Form (Asbestos Certification)**
- Required for nearly all demolitions
- Filed separately with DOB Asbestos Unit
- Must be completed before permit issuance
- See Part 3 for full ACP-5 requirements

**Utility Disconnection Notices**
- Filed with Con Edison (electric, gas)
- Filed with NYC Department of Environmental Protection (water/sewer)
- Must have filed proof before DM permit issues
- See Part 3 for procedures

**DOB NOW Filings - System Navigation**

Steps in DOB NOW:
1. Create applicant account (if new user)
2. Search property by address or block/lot
3. Select "Demolition" as job type
4. Complete DM-1 form within system
5. Upload supporting documents (PDF format)
6. Select payment method (credit card, ACH)
7. Submit for plan examination
8. Monitor status via DOB NOW portal

**BIS Filings - Paper/Digital Hybrid**

Steps in BIS (legacy):
1. Prepare complete hard copy package
2. Obtain BIS filing receipt number
3. Submit at DOB office location or by mail
4. Track through plan examination telephone line
5. [VERIFY] Coordinate any special historic requirements with BIS staff

## Part 3: Pre-Demolition Requirements

### Asbestos Abatement (ACP-5 Certification)

**Why ACP-5 is Required:**
- NYC buildings constructed before ~1980 commonly contain asbestos
- Mechanical removal of asbestos is a separate licensed trade
- Asbestos must be abated before structural demolition begins
- DOB will not issue demo permit without ACP-5 completion or exemption

**ACP-5 Form Overview:**
- Official name: "ACP-5: Declaration of Asbestos Abatement by Asbestos Contractor"
- Filed with DOB's Asbestos Unit (separate from main DOB office)
- Must be filed by a licensed Asbestos Abatement Contractor
- Provides proof that all asbestos has been properly removed/abated

**Process Timeline:**
1. **Phase 1 (0-2 weeks):** AHERA Survey
   - Licensed asbestos inspector conducts Phase I survey
   - Creates list of all ACM (Asbestos Containing Materials)
   - Provides preliminary cost and timeline estimates

2. **Phase 2 (2-8 weeks):** Abatement
   - Licensed abatement contractor performs removal under DOB supervision
   - Notifications to surrounding properties required
   - ACM properly containerized and transported to licensed facility
   - Ongoing air monitoring conducted

3. **Phase 3 (8+ weeks):** ACP-5 Filing
   - Post-abatement clearance air sample testing required
   - Asbestos contractor completes ACP-5 form
   - Filed with DOB with clearance test results
   - DOB reviews and approves (typically 1-2 weeks)

**Cost & Timeline Implications:**
- [VERIFY] Asbestos abatement typically costs $50,000-$500,000+ depending on building size
- Timeline is the critical path item - often 8-12 weeks minimum
- **Start asbestos abatement immediately after initial permit decision**
- DM permit cannot issue until ACP-5 is approved

**ACP-5 Exemptions:**
Very limited exemptions available:
- Applicant certifies building constructed after 1980 (rare)
- Pre-demolition survey shows no ACM present (requires documented inspection)
- [VERIFY] Certain buildings with prior abatement documentation
- **Default assumption:** Any building built before 2000 requires full ACP-5

**Documentation Required:**
- Phase I AHERA Survey report
- Licensed inspector credentials
- Abatement contractor license verification
- Clearance air sample test results (lab certified)
- ACP-5 form signed by abatement contractor and inspector

### Utility Disconnections and Notifications

**Why Utility Work is Mandatory Pre-Demo:**
- NYC law requires all utilities serving building to be discontinued before demolition
- Emergency protocols exist but only when utility company confirms disconnection impossible
- Both service disconnections AND abandonments required for different utilities
- This is a critical path item - must start early

**Con Edison (Electric and Gas Disconnections)**

**Process:**
1. **Notify Con Ed of Intent (60 days prior)**
   - Call Con Ed's Commercial Services: [VERIFY - current phone number]
   - Provide address and expected demolition date
   - Con Ed assigns project manager

2. **Service Disconnection Application**
   - Obtain Con Ed Form #[VERIFY] for disconnection request
   - Provide proof of ownership/control
   - Specify disconnection vs. abandonment (both required)
   - Submit 30-45 days before intended disconnection date

3. **Field Visit and Inspection**
   - Con Ed inspector visits site
   - Verifies building status and safety conditions
   - Assesses equipment condition and location
   - Identifies any operational/debt issues

4. **Disconnection Execution**
   - Con Ed cuts service lines outside property (external cut)
   - Building interior lines abandoned in place (in-wall lines left as-is)
   - Con Ed provides "Discontinuance of Service" letter
   - **Obtain and retain this letter for DOB filing**

**Timeline:** 8-12 weeks minimum from initial notification

**NYC Department of Environmental Protection (Water/Sewer)**

**Process:**
1. **Water Service Disconnection**
   - File disconnection request with DEP Customer Services
   - [VERIFY - current procedures] May require separate application or contractor notification
   - Provide address and target disconnection date
   - DEP typically disconnects at street connection point

2. **Sewer Connection Abandonment**
   - Filed with DEP's Sewer Connection Removal Program
   - Different from water service - rarely involves physical disconnection
   - Usually certification that connection will be sealed during demo

3. **Stormwater/Roof Drains**
   - If building has dedicated stormwater discharge, must be addressed
   - May require street connection capping
   - [VERIFY - current procedures with DEP]

**Timeline:** 4-8 weeks minimum

**Telecommunications (Verizon/Cable Operators)**

**Process:**
1. Notify primary telecom provider serving building
2. Request service disconnection (less regulated than utilities)
3. Obtain disconnection confirmation letter
4. [VERIFY] If multiple carriers exist, each must be notified

**Timeline:** 2-4 weeks

**Critical Documentation for DOB Filing:**
Before submitting DM permit, obtain:
- Con Edison Discontinuance of Service letter (electric)
- Con Edison Discontinuance of Service letter (gas)
- DEP Water Service Disconnection confirmation
- DEP Sewer Connection documentation
- Telecom disconnection confirmation (if applicable)
- **Keep originals in permit file**

**Common Delays with Utilities:**
- Con Ed accounting disputes or prior owner debt issues
- Property access issues preventing meter location verification
- High-voltage transmission lines requiring specialized coordination
- Multiple service connections from different eras requiring separate disconnections
- [VERIFY] Circumstances where utility company cannot disconnect (rare emergency cases)

### DOB Pre-Demolition Notifications

**Superintendent of Schools (S.O.S.) Certification**

**Requirement:**
- NYC Education Law § 4407 mandates notice to School District
- Required for all demolitions, regardless of proximity to schools
- DOB will not accept DM-1 without S.O.S. certification

**Process:**
1. Contact NYC Department of Education, Office of School Construction
2. Provide building address and target demolition date
3. DOB submits form to School District on behalf of applicant
4. [VERIFY - current timeline] S.O.S. response typically 10-15 days
5. Response attached to DM-1 application

**Documentation Required:**
- S.O.S. Certification letter confirming notice was given
- Confirmation that DOB notified appropriate School District
- Retain for permit file

**Pre-Demolition Survey and Notification (for adjacent buildings)**

See Part 5 (Neighboring Buildings) for full NB-2 survey procedures.

## Part 4: Site Safety Requirements

### Site Safety Manager (SSM) Designation

**Legal Requirement:**
- New York State Industrial Code § 12-NYCRR 23-1.20
- Required for all demolition work in NYC
- SSM must be on-site continuously during demolition
- Different from Site Safety Coordinator (see below)

**SSM Qualifications:**
- Licensed New York State Professional Engineer or Architect, OR
- Individual with documented 5+ years demolition supervision experience
- [VERIFY - current credential requirements with DOB]
- Must pass DOB written examination (if experience-based route)
- Must maintain current insurance

**SSM Responsibilities:**
- Daily pre-demolition safety inspections
- Continuous presence during active demolition work
- Coordination with Site Safety Coordinator (SSC)
- Authority to stop unsafe work
- Daily written safety logs
- OSHA compliance oversight
- Incident reporting to DOB

**Designation Process:**
1. Identify qualified SSM candidate
2. Verify credentials and insurance
3. Complete Form SSM-1 (Designation of Site Safety Manager)
4. SSM signs form confirming acceptance
5. Attach to DM-1 permit application
6. [VERIFY] File copy with DOB prior to permit issuance

### Site Safety Coordinator (SSC) - When Required

**When SSC is Required vs. Optional:**

**REQUIRED for:**
- Partial demolitions where neighboring buildings within 25 feet exist
- Buildings in dense urban areas (typically Manhattan, parts of Brooklyn)
- Multi-story demolitions (3+ stories)
- Buildings adjacent to occupied structures
- Demolitions within 50 feet of active businesses

**OPTIONAL for:**
- Full demolitions of standalone buildings in less-dense areas
- Single-story structures with no adjacent occupied buildings
- Rural or industrial areas with significant setbacks

**SSC Qualifications:**
- Licensed Structural Engineer or Architect in New York State, OR
- [VERIFY] Minimum 3 years demolition-specific structural oversight
- Must carry errors & omissions insurance
- Different from SSM - focuses on structural protection, not active demolition

**SSC Responsibilities:**
- Pre-demolition condition survey (photographic/written documentation)
- Development of neighboring building protection plan
- Structural monitoring during demolition
- Certification that neighboring buildings are protected
- Authority to require work stoppage if adjacent building at risk
- Final post-demolition inspection certification

**Designation Process:**
1. Identify qualified SSC candidate
2. Verify structural engineering license (if applicable)
3. Complete Form SSC-1 (Designation of Site Safety Coordinator) [VERIFY - form name]
4. Attach to DM-1 permit application
5. SSC prepares pre-demolition survey documentation

### Site Safety Plan and CEQR

**Site Safety Plan Components:**

All demolition permits require submission of a site-specific safety plan addressing:

1. **Boundary and Access Control**
   - Perimeter fencing specifications
   - Access gates and controls
   - Signage and warnings
   - Hours of operation

2. **Utility Disconnections**
   - Electrical disconnection procedures
   - Gas disconnection procedures
   - Water/sewer handling
   - Telecom line management

3. **Environmental Protections**
   - Dust control measures (water trucks, barriers)
   - Noise limitations and hours
   - Vibration monitoring (if applicable)
   - Stormwater pollution prevention (SWPPP)

4. **Debris and Waste Handling**
   - Demolition debris disposal plan
   - Hazardous material handling
   - Recycling/salvage plans
   - Transportation routes and hours

5. **Neighboring Building Protections**
   - Shoring specifications
   - Monitoring requirements
   - Emergency procedures
   - Contact information

6. **Emergency Procedures**
   - Emergency contact list
   - Incident reporting protocols
   - Evacuation procedures
   - Medical emergency response

7. **Inspection Schedule**
   - DOB inspection frequency (typically 1x/week minimum for active demo)
   - Third-party monitoring (if required)
   - Structural monitoring (for adjacent buildings)
   - Final closeout inspection requirements

**CEQR (City Environmental Quality Review)**

**When CEQR is Required:**
- [VERIFY] Most demolitions trigger CEQR requirements
- Determination made by relevant NYC agency (typically DOB)
- CEQR is often incorporated into DM permit review

**CEQR Assessment Includes:**
- Environmental impact on adjacent properties
- Traffic and transportation impacts
- Air quality impacts (dust, emissions)
- Noise impacts and mitigation
- Hazardous materials assessment
- Community impacts

**Documentation Required:**
- CEQR form or statement (if required by DOB)
- Environmental conditions assessment
- Mitigation measures documentation
- [VERIFY] Agency determination letter

## Part 5: Neighboring Building Protections

### NB-2 Survey Requirements

**When NB-2 is Required:**
- Any building adjacent to demolition (typically within 25 feet)
- Multi-story demolitions with adjacent structures
- Buildings with shared party walls
- Partial demolitions on lots with multiple structures
- Any excavation work that may affect adjacent foundations

**NB-2 Process Overview:**
Pre-demolition condition documentation protecting adjacent building owners' rights.

**Step 1: Identify Adjacent Buildings**

Define adjacency by:
- Distance from property line: typically 25-50 feet
- Shared walls or foundations
- Potential impact from vibration or settlement
- Direct line-of-sight from demolition site

**Common Adjacent Building Scenarios:**
- Side-by-side buildings on NYC block (very common in Manhattan)
- Buildings connected by shared wall party
- Buildings with basements extending under street/alley
- Buildings with cantilevered structures over neighbor property

**Step 2: Pre-Demolition Survey (NB-2 Form)**

**Survey Components:**
1. **Property Identification**
   - Neighboring building full address
   - Block/lot number
   - Building condition assessment

2. **Building Description**
   - Age and construction type (masonry, steel frame, etc.)
   - Current use and occupancy status
   - Visible foundation/basement configuration
   - Party wall conditions (if shared wall exists)

3. **Photographic Documentation**
   - Minimum 6-8 photos per adjacent building
   - Facade conditions, visible cracks or damage
   - Foundation/basement areaway conditions
   - Party wall conditions (interior if accessible)
   - Roof conditions and structural support
   - Utility connections visible at building

4. **Written Condition Certification**
   - Professional assessment of structural condition
   - Identification of pre-existing damage or deterioration
   - Baseline establishment for post-demolition comparison
   - Signed affidavit by qualified professional [VERIFY - required credential]

**Step 3: NB-1 Affidavit (Proof of Service)**

**Purpose:**
Demonstrates that adjacent building owner was notified of impending demolition and had opportunity to request protective measures.

**Process:**
1. Prepare notice letter to adjacent building owner(s)
   - Include property address being demolished
   - Include planned demolition date
   - Include contact information for demolition contractor
   - Include request for inspection agreement/waiver

2. Serve notice by:
   - Personal delivery to owner/occupant (preferred)
   - Certified mail to owner's last known address
   - Posted on building if owner not accessible
   - Service to property manager or agent if available

3. Document proof of service
   - Mail receipts for certified letters
   - Affidavit of personal service (if applicable)
   - Photos of posted notices (if applicable)
   - Complete NB-1 Affidavit sworn before notary public

4. Attach to DM permit with supporting documentation

**Cost of NB Protections:**
- Survey/documentation: $2,000-$5,000 per adjacent building
- Protective measures (shoring, bracing): $10,000-$100,000+
- Monitoring/inspections during demo: $5,000-$20,000
- Post-demolition restoration (if damage occurs): highly variable

### Shoring and Protection Plans

**When Shoring is Required:**

1. **Party Wall Demolitions**
   - Shared wall with adjacent building must be permanently braced or demolished with building
   - Cannot demolish building and leave party wall (creates hazard)
   - Options: (a) Install permanent shoring posts, or (b) Demolish party wall simultaneously
   - Licensed structural engineer must design system

2. **Close Adjacency (0-10 feet)**
   - Temporary bracing during demolition standard
   - Dust suppression and vibration monitoring required
   - Specialized demolition sequencing to minimize impact
   - Continuous structural engineer monitoring

3. **Foundation/Basement Impacts**
   - Basements extending under demolition site require special procedures
   - [VERIFY] Cannot undermine adjacent foundations during demo
   - May require ground stabilization or temporary underpinning
   - Engineering assessment mandatory

**Shoring Design Components:**

Specifications prepared by licensed structural engineer:
- Load calculations for wall/foundation bearing
- Post specifications and bearing capacity
- Bracing and lateral support requirements
- Installation sequence and temporary supports
- Monitoring requirements during installation
- Removal procedures and timing
- Inspection and certification protocol

**Plan Submission:**
- Shoring plans submitted with NB-2 survey documentation
- May require separate Building Information Services (BIS) filing [VERIFY]
- Structural engineer stamps and signs all plans
- DOB may require plan examination before permit issuance

### Pre-Demolition Agreement (NB-3)

**Purpose:**
Agreement between demolition applicant and adjacent building owner establishing rights, responsibilities, and remedies related to demolition.

**Typical Provisions:**

1. **Acknowledgment of Demolition**
   - Adjacent owner acknowledges planned demolition
   - Accepts NB-2 baseline condition documentation
   - Waives further notice requirements

2. **Protective Measures**
   - Specifies any shoring or bracing to be installed
   - Identifies monitoring protocols
   - Establishes hours of work restrictions (if any)
   - Noise and dust control requirements

3. **Liability and Insurance**
   - Demolition contractor maintains specified insurance limits
   - Adjacent building owner waives certain claims related to demolition
   - [VERIFY - current standards] Typical waiver limits to normal demolition activities, excludes contractor negligence
   - Insurance requirements documented

4. **Dispute Resolution**
   - Process for addressing damage claims
   - Inspection procedures and cost allocation
   - Third-party neutral/mediator procedure
   - Timeline for claim presentation

5. **Restoration/Repair**
   - Responsibility for fixing demolition-caused damage (vs. pre-existing)
   - Restoration timeline
   - Cost allocation
   - Quality standards and contractor selection

**Negotiation Considerations:**
- Adjacent owners often request higher protections/monitoring
- May demand continuous structural engineer on-site (cost: $200-500/day)
- Some owners request escrow for potential damage claims
- [VERIFY - current practice] Settlement amounts typically $5,000-$50,000 depending on building value
- Legal counsel advisable for complex properties

## Part 6: Mechanical vs. Hand Demolition Thresholds

### Mechanical Demolition Standards

**Definition:**
Use of heavy equipment (excavators, backhoes, loaders, impactors) to demolish structure.

**Mechanical Demolition Permitted For:**
- Buildings exceeding 25 feet in height (3+ stories typically)
- Structures with frame construction exceeding 5,000 square feet
- Buildings with concrete or steel primary structure
- Industrial or commercial structures
- Buildings where structural complexity allows machine operation

**Equipment Specifications:**
- Excavator with hydraulic hammer or breaker bucket
- Typical equipment rental: $15,000-30,000/month
- Operator certification: [VERIFY - current OSHA/NYS requirements]
- Insurance: General Contractor should carry equipment liability

**Safety and Oversight:**
- Continuous SSM monitoring required
- Dust suppression equipment (water truck or spraying system)
- Vibration monitoring on adjacent structures
- Power line clearance verification (minimum 10 feet clearance for 480V lines; greater for higher voltages)
- Weekly DOB inspections standard

**Limitations on Mechanical Demo:**
- Cannot operate within certain proximity to adjacent buildings [VERIFY - exact setback/distance requirements]
- Restricted hours in residential areas (often 7am-6pm, Mon-Fri)
- May require additional permits for street occupancy if equipment extends over sidewalk
- Environmental permits may be required for special circumstances

### Hand Demolition Standards

**Definition:**
Manual removal using hand tools, pry bars, sledgehammers, and other non-powered equipment.

**Hand Demolition Required For:**
- Buildings within 25 feet of occupied structures with no protective agreement
- Historic buildings with preservation requirements
- Structures with delicate adjacent neighbors (banks, offices with sensitive equipment)
- Controlled demolition in dense urban areas
- Small structures (sheds, carriage houses, garages)
- Buildings where mechanical equipment cannot access

**Hand Demo Process:**
1. Manual dismantling from top-down
2. Materials lowered by crane or drop chutes to ground level
3. Segregation of materials for recycling/salvage
4. Slower process: typically 6-12 months vs. 2-4 months mechanical
5. Higher labor costs, lower equipment costs

**Labor and Cost Implications:**
- Labor cost: $100,000-$500,000+ depending on building size
- Timeline significantly longer (300% to 500% longer than mechanical)
- Material salvage often makes economics feasible
- Appropriate for historic districts or premium material buildings

**Hybrid Approach:**
Many demolitions use combination:
- Interior demolished mechanically
- Exterior facade preserved or hand-demolished
- Structural frame mechanically removed
- Interior finishes hand-segregated for salvage

### Thresholds and Decision Factors

**Decision Tree for Method Selection:**

| Factor | Mechanical Demo | Hand Demo |
|--------|-----------------|-----------|
| **Building Height** | 3+ stories | 1-2 stories |
| **Distance to Neighbor** | 25+ feet (or protected by agreement) | <25 feet (unprotected) |
| **Structure Type** | Steel/concrete frame | Masonry/wood |
| **Adjacent Occupancy** | Commercial or vacant | Occupied (sensitive use) |
| **Historic Designation** | None | Landmark or in district |
| **Site Conditions** | Adequate equipment access | Limited access |
| **Timeline** | 8-12 weeks | 6-12 months |
| **Budget** | Equipment + Labor | Labor-intensive |

**DOB Inspection Frequency Based on Method:**

- **Mechanical Demo:** Weekly minimum (DOB inspector visits site)
- **Hand Demo:** Typically 2x weekly during active phases
- **High-risk locations:** Daily inspections possible
- **Emergency Demo:** As-needed inspection schedule

## Part 7: Emergency Demolition Procedures

### When Emergency Demo Applies

**NYC Administrative Code § 27-305 Requirements:**

Emergency demolition authorized when:
1. Building poses imminent danger to public safety
2. Ordinary demolition procedures cannot be followed due to urgency
3. Building collapse appears likely
4. Structural failure evident and spreading
5. Hazardous material exposure risk

**Examples Triggering Emergency Demo:**
- Building partially collapsed
- Structural fire damage with failure risk
- Severe foundation failure or subsidence
- Exterior facade or wall falling from building
- Hazardous material leak (asbestos, chemicals) threatening public
- Natural disaster damage (hurricane, earthquake) causing imminent danger

### Emergency Demo Expedited Process

**Step 1: DOB Emergency Declaration (24-48 hours)**

Procedure:
1. Contact DOB Emergency Services (24-hour hotline) [VERIFY - current number]
2. Provide building address and description of emergency condition
3. Request DOB emergency inspector assignment
4. Inspector visits site same day or within 24 hours
5. Inspector determines if emergency demolition warranted
6. If approved, DOB issues Emergency Demolition Authorization letter

**Step 2: Emergency Demo Permit Filing (Simultaneous with Declaration)**

Expedited application:
1. File DM-1 application with "EMERGENCY" clearly marked in header
2. Annotate application: "Filed pursuant to NYC Administrative Code § 27-305 Emergency Demolition"
3. Submit minimal supporting documentation initially:
   - Applicant contractor license copy
   - Emergency declaration photo documentation
   - DOB inspector's authorization letter
4. ACP-5 asbestos documentation [VERIFY - requirements; likely waived or expedited in true emergency]
5. Insurance documentation

**Step 3: Permit Issuance (24-72 hours)**

Compressed timeline:
- Plan examination condensed or waived for true emergencies
- Permits issued verbally over phone, followed by written documentation
- All standard permit conditions still apply
- Emergency permits still require SSM designation

**Step 4: Post-Emergency Documentation (Within 30 Days)**

After emergency demo begins/completes:
1. Submit all standard documentation (ACP-5, utility discontinuances, surveys)
2. Document emergency justification with photos/report
3. Provide weekly updates to DOB
4. Final completion documentation

### Emergency Demo Exemptions and Waivers

**Potential Exemptions in True Emergency:**
- [VERIFY] ACP-5 may be expedited or waived if utility disconnection impossible
- Utility disconnection timelines may be shortened or waived
- NB-2 survey and neighboring building protections may be expedited (but not eliminated)
- CEQR review may be streamlined

**Limitations - What Cannot Be Waived:**
- SSM designation still required
- Insurance/bonding still required
- Environmental protections still apply
- Neighboring building safety cannot be completely ignored
- Hazardous material handling must follow all procedures

**Post-Emergency Compliance:**
- All normal procedures must be satisfied within 30-60 days
- ACP-5 abatement must be completed even if expedited initially
- Utility companies still must formally disconnect services
- DOB may impose special conditions on emergency demo permits

## Part 8: Insurance and Bonding Requirements

### General Liability Insurance

**Minimum Coverage Required:**

Standard demolition projects:
- **Limit per occurrence:** $1,000,000
- **General aggregate limit:** $2,000,000
- **Products/Completed operations:** $1,000,000

High-risk or large demolitions:
- [VERIFY - current DOB standards] May require $2,000,000 per occurrence
- Adjacent occupied buildings: often $3,000,000+ demanded in NB agreements
- Downtown Manhattan projects: sometimes $5,000,000+ per occurrence

**Policy Requirements:**

1. **Named Insured**
   - General Contractor (demolition applicant)

2. **Additional Insured Endorsements**
   - NYC Department of Buildings (required)
   - City of New York (required in some cases)
   - Property owner/applicant
   - Adjacent building owners (often required by NB agreement)

3. **Waiver of Subrogation**
   - Insurance cannot pursue claims against adjacent building owners
   - Protects adjacent property owners from insurer claims

4. **Coverage Scope**
   - Bodily injury (third party)
   - Property damage (third party)
   - Products liability (if any salvaged materials sold)
   - Demolition operations specifically included

### Critical Path Analysis

**Asbestos Abatement is the Longest Lead Item:**
1. Must identify qualified asbestos contractor (1-2 weeks)
2. Scope survey and abatement plan (1-2 weeks)
3. Abatement execution (4-10 weeks)
4. Post-abatement clearance testing and ACP-5 filing (1-2 weeks)
5. **Total: Minimum 8 weeks, often 12+**

**Action Item for Expediter:**
- Initiate asbestos screening immediately upon client decision to demolish
- Do not wait for permit application approval to start abatement
- Parallel track: start asbestos work while preparing permit package

**Utility Disconnections Second-Longest Lead:**
1. Con Edison notification and application (6-8 weeks)
2. Con Edison scheduling and execution (typically within 2-3 weeks of application)
3. DEP processes (4-6 weeks)
4. **Total: 8-12 weeks minimum**

**Action Item for Expediter:**
- Contact utilities immediately after property control confirmed
- Provide all requested information immediately to accelerate processing

### Common Delays and Mitigation Strategies

**Delay #1: Incomplete ACP-5 Documentation**

Cause:
- Phase I survey incomplete or unclear
- Clearance test results delayed or failed
- Asbestos contractor license expired or invalid

Mitigation:
- Verify asbestos contractor license immediately (not at end of project)
- Monitor Phase I survey scope carefully (insist on completeness)
- Budget for retesting if initial clearance fails
- Expect 2-4 weeks for clearance testing/retesting

**Delay #2: Con Edison Service Issues**

Cause:
- Prior owner debt or billing disputes preventing disconnection
- Multiple service connections from different eras
- Meter location discovery requiring extensive investigation
- High-voltage transmission lines requiring special protocols

Mitigation:
- Obtain title clearance confirming no prior owner liens
- Contact Con Ed very early to identify potential issues
- Expect Con Ed to require site visit and verification (budget 1 month)
- Get written service disconnection estimate and timeline early

**Delay #3: NB-2 Survey Access Issues**

Cause:
- Adjacent building owner refuses access for survey
- Tenants in adjacent building prevent inspection
- Owner demands payment before granting access

Mitigation:
- Provide formal service notice early (see NB-1 process)
- Offer to conduct exterior survey only if interior inaccessible
- Coordinate with adjacent building owner's counsel if agreement needed
- Budget 3-4 weeks for NB-2 completion if access contested

**Delay #4: DOB Plan Examination Deficiencies**

Cause:
- Incomplete site safety plans
- Structural engineer's shoring plans inadequate
- Environmental assessment missing required elements
- Multiple deficiencies requiring multiple resubmittals

Mitigation:
- Have structural engineer and safety planner review docs before filing
- Submit most complete application possible (reduces deficiency rounds)
- Respond to deficiencies within 5 business days (not 10)
- Request expedited plan examination if project time-critical

**Delay #5: Historic Preservation/Landmarks Review**

Cause:
- Building subject to Landmarks Preservation Commission (LPC) designation
- LPC Permit (LP) required before DOB demolition permit can issue
- LPC approval process can take 2-4 months

Mitigation:
- Verify LPC designation before starting process
- If LPC-designated, file LP application parallel to (or before) DM application
- Coordinate with LPC staff early for guidance
- Budget 4-6 weeks additional timeline for LPC approval

**Delay #6: Neighboring Building Issues**

Cause:
- Adjacent building owner demands protective measures not in initial plan
- NB agreement negotiations protracted
- Structural engineer's shoring plan requires redesign

Mitigation:
- Identify adjacent building requirements early
- Engage NB owner's counsel proactively
- Budget for enhanced protections (shoring, monitoring) if necessary
- Allow 4-6 weeks for NB agreement negotiation

**Delay #7: Environmental Contamination Discovery**

Cause:
- Phase I environmental assessment reveals hazardous materials (groundwater contamination, underground storage tanks, soil issues)
- Requires additional environmental permits
- May require specialized abatement contractor
- Adds months to project timeline

Mitigation:
- Commission Phase I environmental assessment before financing project
- Budget for Phase II (soil testing) if Phase I indicates concerns
- Establish remediation budget and timeline early
- Coordinate with NYC Department of Environmental Protection

   - If uncertain about requirement, provide extra detail (better to over-document than under-document)

**Typical Outcome:**
- Plan examination deficiencies common (30-50% of applications) and expected
- Most deficiencies resolved in 1-2 resubmission rounds
- Total timeline impact: 3-4 weeks if deficiency response quick
- Causes delay but rarely results in permit denial

**Objection: "Environmental contamination found - work must stop immediately"**

**Why It Arises:**
- Unexpected contaminated soil discovered during demolition
- Groundwater contamination encountered during excavation
- Asbestos or hazardous materials found despite pre-demolition survey
- Community member or environmental agency alerts DOB to environmental concern

**Resolution Strategies:**

1. **Regulatory Notification (Required)**
   - Notify NYC DEP immediately if soil/groundwater contamination suspected
   - Notify DOB if work must pause
   - Do NOT try to remediate without regulatory approval
   - Document discovery (photos, location description, extent)

2. **Environmental Professional Engagement**
   - Hire environmental consultant to assess contamination scope
   - Obtain Phase II testing results (soil samples, analysis)
   - Determine remediation approach and cost
   - Get regulatory approval for remediation plan

3. **Regulatory Approval for Remediation**
   - File remediation plan with NYC DEP
   - Obtain DEP approval before remediation commences
   - [VERIFY - current procedures] May require specialized licensed contractor
   - Timeline: 4-12 weeks for approval plus remediation duration

4. **Project Impact**
   - Work stoppage: 1-2 weeks during assessment
   - Regulatory approval: 4-8 weeks
   - Remediation: 4-12 weeks depending on contamination extent
   - **Total delay: 3-6 months minimum**
   - **Cost impact: $50,000-$500,000+ depending on contamination severity**

5. **Financial and Insurance Considerations**
   - Environmental liability insurance may cover remediation costs
   - Contractor may not be responsible if pre-existing condition
   - Owner/developer ultimately bears cost
   - Advocate with owner for budget/timeline adjustment

**Typical Outcome:**
- Environmental issues rare (5-10% of projects in industrial areas)
- Assessment and remediation significantly delays project
- Regulatory process must be followed; no shortcuts available
- Budget contingency of 15-20% of demo cost advised for industrial properties

---

**Objection: "Insurance expires during project - renewal required"**

**Why It Arises:**
- General Liability insurance policy expires during demolition phase
- Renewal certificate not provided to DOB
- Coverage lapse creates permit violation

**Resolution Strategies:**

1. **Insurance Timeline Management**
   - At project start, identify insurance policy expiration date
   - Notify contractor/broker of renewal deadline (6-8 weeks before expiration)
   - Request renewal certificate 4 weeks before expiration
   - DOB must receive renewed certificate before expiration

2. **Certificate of Insurance Update**
   - Obtain renewed Certificate of Insurance from broker
   - Verify DOB and property owner still listed as additional insureds
   - Submit to DOB or retain in permit file (depending on DOB requirement)
   - Communicate renewal to Site Safety Manager and site supervisor

3. **Coverage Continuation Options**
   - Continuous coverage: Renew policy 30+ days before expiration (preferred)
   - Endorsement extension: If renewal delayed, insurer can issue temporary endorsement extending coverage 30 days (short-term fix)
   - New policy: If change of carrier necessary, ensure new policy starts before old one expires

4. **If Coverage Lapse Occurs**
   - Notify DOB immediately
   - Obtain emergency certificate of insurance
   - May require stop-work order until coverage verified
   - Avoid coverage lapses at all costs (permit violation)

**Typical Outcome:**
- Insurance renewal routine; rarely causes significant delay if planned ahead
- Cost: Renewal premium typically same as initial (included in contractor's budget)
- Simple administrative task requiring early attention

## Summary: Expediter Workflow Checklist

### Pre-Filing (4-12 Weeks Before DM Application)

- [ ] Verify property control (deed, authorization letter, or lease)
- [ ] Check LPC designation status (Landmarks Preservation Commission)
- [ ] Initiate Phase I environmental site assessment (if industrial property)
- [ ] Start asbestos screening survey immediately
- [ ] If asbestos abatement needed, contract licensed abatement firm and begin Phase II
- [ ] Notify Con Edison, DEP of demolition intent (early notification = faster processing)
- [ ] Identify Site Safety Manager candidate; verify qualifications and insurance
- [ ] Assess need for Site Safety Coordinator (adjacent buildings?)
- [ ] If adjacent buildings exist, identify owners and plan NB engagement strategy
- [ ] Obtain contractor license verification (General Contractor with Demolition Classification)
- [ ] Arrange General Liability insurance ($1M+ GL); verify DOB as additional insured
- [ ] Obtain all utility disconnection letters (Con Ed, DEP)
- [ ] Compile proof of site control documentation

### Permit Application (2 Weeks Before Filing)

- [ ] Prepare DM-1 application with all required fields
- [ ] Compile NB-2 survey (if applicable)
- [ ] Prepare site safety plan document
- [ ] Have structural engineer review shoring/protection plans (if applicable)
- [ ] Prepare SSM-1 form with qualified Site Safety Manager signature
- [ ] Verify ACP-5 approval (or prepare exemption affidavit)
- [ ] Obtain S.O.S. (Superintendent of Schools) certification
- [ ] Obtain final utility disconnection confirmation letters
- [ ] Collect all insurance and bonding documentation
- [ ] Calculate all DOB fees and prepare payment

### Filing and Plan Examination (On Filing)

- [ ] Submit complete DM-1 package via DOB NOW (or BIS, if required)
- [ ] Monitor DOB NOW status portal for plan examination updates
- [ ] If deficiency notice received: respond within 5 business days
- [ ] Obtain final permit from DOB (verbal approval confirmed in writing)
- [ ] Receive written permit document with conditions and effective date

### Active Demolition (During Work)

- [ ] Ensure SSM on-site before first day of work
- [ ] Facilitate DOB pre-demolition inspection (24-48 hours before work starts)
- [ ] Maintain organized permit file on-site with all documentation
- [ ] Monitor DOB weekly inspections; address any "Notice to Correct" items
- [ ] If SCC in place, ensure weekly structural monitoring reports submitted to DOB
- [ ] Keep adjacent building owners informed of progress (if applicable)
- [ ] Maintain insurance certificate currency throughout project
- [ ] Document incident reports and safety concerns for DOB file

### Project Closeout (After Demolition Complete)

- [ ] Request and obtain final DOB inspection
- [ ] Ensure all debris removed from site
- [ ] Verify final utility disconnections confirmed by utilities
- [ ] Obtain final SSC certification (if applicable)
- [ ] Archive all permit file documentation for [VERIFY] retention period
- [ ] Final DOB completion/permit closure
