# ALT-1 (Alteration Type 1) Filing Guide

## Overview

ALT-1 is the filing designation for **major alterations** — work that triggers specific regulatory thresholds that require full DOB review and approval. Unlike minor alteration filings, ALT-1 work involves changes that materially affect the building's use, occupancy, egress, or structural integrity.

### What Qualifies as ALT-1

ALT-1 applications are required when alterations involve:

- **Change of Use or Occupancy Class** — Converting between occupancy groups (e.g., commercial to residential, office to retail, etc.)
- **New Egress Routes** — Adding, relocating, or modifying exit pathways; changes to stair widths or travel distances
- **Structural Modifications** — Work affecting the primary structural system (columns, beams, load-bearing walls, floor systems)
- **Legalization of Existing Conditions** ("No-Work" ALT-1) — Formalizing non-compliant conditions without physical alteration
- **Dwelling Unit Reconfiguration** — Combining, splitting, or converting units in residential buildings

**Common ALT-1 Project Types:**
- Hotel-to-residential conversions
- Commercial space to residential apartments
- Legalization of existing dwelling units or occupancy changes
- Addition of new floors
- Substantial structural reinforcement

## ALT-1 vs. ALT-2 vs. PAA: Filing Decision Tree

Use this table to determine the correct filing type:

| Filing Type | Use Case | Key Characteristics |
|-------------|----------|-------------------|
| **ALT-1** | Major alterations | Change of use/occupancy, new egress, structural changes, legalization |
| **ALT-2** | Minor alterations | Multiple work types (trades), no occupancy change, no egress mods, no primary structural work |
| **PAA** | Assembly spaces | Any assembly use with ≥75 occupants OR assembly with food service |

**Decision Rule:** If your work involves occupancy change, new egress, or structural modifications to primary members, file ALT-1 first. If it's interior renovation with multiple trades but no occupancy change, consider ALT-2. If creating an assembly space, file both ALT-1 (for construction) and PAA (for assembly certification).

## Required Documents Checklist

Before submitting, ensure you have:

- [ ] **PW1 Form** (Alteration Application, filed digitally in Build)
- [ ] **Architectural Plans** — Existing conditions (as-built) + proposed conditions; all affected floors; detail sheets for modified areas
- [ ] **Structural Plans** — If structural work involved; certified by PE
- [ ] **MEP Plans** — Mechanical, electrical, plumbing plans if those systems are affected
- [ ] **TR1 Form** — Energy code compliance; may require EN (Energy Code Worksheets) sheets
- [ ] **Cost Affidavit** — Signed estimate of total construction cost
- [ ] **Owner's Authorization** — Property owner authorization/consent letter
- [ ] **Applicant's Seal** — Licensed PE or RA seal on relevant sheets
- [ ] **Tenant Protection Plan (TPP)** — If occupied dwelling units exist during construction
- [ ] **Zoning Documentation** — Zoning analysis confirming proposed use is permitted
- [ ] **Fire Protection Submittals** — FDNY filings if triggering sprinkler, fire alarm, or standpipe work

## Common ALT-1 Filing Scenarios

### Scenario 1: Change of Occupancy (Commercial to Residential)
Typical path: Office building → residential apartments

**Why ALT-1?** Occupancy change requires full egress redesign, stairwell sizing verification, unit configuration plans, energy code compliance.

**Additional reviews:** Zoning (residential permitted?), NYC Housing Preservation Department if applicable, egress analysis.

### Scenario 2: Legalization ("No-Work" ALT-1)
Converting existing unpermitted dwelling units to legal status without construction.

**Why ALT-1?** Creates formal record; triggers occupancy classification.

**Note:** Plans must still show existing conditions accurately. [VERIFY: Can this be filed simultaneously with ALT-2 for other work?]

### Scenario 3: Hotel/Residential Conversion
Converting hotel guest rooms to permanent apartments.

**Why ALT-1?** New occupancy class, potential egress modifications, zoning implications.

**Special considerations:** Verify zoning allows permanent residential use; coordinate with HPD if required.

### Scenario 4: Unit Combination/Splitting
Removing/adding walls between residential units to reconfigure layout.

**Why ALT-1?** Changes number of dwelling units; may affect egress, mechanical distribution.

### Scenario 5: Adding New Floors
Structural addition to existing building.

**Why ALT-1?** Primary structural modification.

## Plan Examination Process

### Submission
File PW1 and all supporting documents in DOB Build. System generates filing number and assigns to examination queue.

### Plan Review
DOB assigns an examiner from the ALT1 division. Review period: [VERIFY: Typical turnaround for first-round review? 30 days?]

### Objections
Examiner issues objections (deficiencies) via DOB Build. Common objections:
- Egress non-compliance (travel distance, stair width, path of travel)
- Energy code deficiency (TR1 inadequate)
- Zoning violation (use not permitted, FAR exceeded, etc.)
- Fire/life safety (fire ratings, separation, sprinkler requirements)
- Occupancy load disputes
- Structural inadequacy

## "No-Work" ALT-1: Legalization Without Construction

A "no-work" ALT-1 formalizes existing conditions without physical alteration. Common uses:
- Legalizing unauthorized dwelling units already in building
- Changing occupancy classification of existing space
- Bringing non-compliant conditions into the permit record

**Key Point:** Even though no construction occurs, you must submit plans showing existing conditions truthfully. This creates the legal record.

## TPP (Tenant Protection Plan)

If dwelling units are **occupied during construction**, a Tenant Protection Plan is required.

**TPP Covers:**
- Temporary heat/hot water
- Noise/dust control
- Access to services and exits
- Relocation assistance if needed
- Compliance timeline

File TPP in Build as part of ALT-1 submission if occupied units exist.

## Final Steps to Completion

### Final Inspection
Once all construction complete, request Final Inspection. DOB inspector verifies work matches approved plans, all violations corrected, inspections passed.

### Letter of Completion or Certificate of Occupancy
- **Letter of Completion** — For interior ALT-1 where occupancy unchanged
- **Certificate of Occupancy** — For ALT-1 involving occupancy change; confirms new use is legal

Obtain from DOB after final inspection approval. **Must post CO at building entrance** before occupancy.

## Common ALT-1 Objections and Solutions

| Objection | Root Cause | Solution |
|-----------|-----------|----------|
| Egress travel distance exceeds code | Room too deep for exit proximity | Add secondary exit, corridor door, or reduce occupant load |
| Stair width insufficient | Sizing not recalculated for new occupancy | Revise stair geometry to meet occupant load |
| Energy code non-compliance (TR1) | Insulation, HVAC, window spec inadequate | Upgrade envelope or HVAC per TR1 calculator |
| Use not permitted by zoning | Proposed occupancy not allowed in district | Seek variance, apply for Use Variance, or change scope |
| Fire rating deficient | Fire-rated assemblies not specified | Add fire-rated drywall, doors, dampers as required |
| Occupancy load incorrect | Load calculation errors | Recalculate using net floor area per occupancy table |
