# Determination Filing Guide

## Overview

A Determination is a formal, written ruling by the NYC Department of Buildings (DOB) that interprets the Building Code, Administrative Code, or other DOB regulations as they apply to a specific property, condition, or project. This guide covers the types of determinations, the filing process, required documentation, and strategic considerations for pursuing determinations in expediting practice.

## What is a Determination?

A Determination is an official DOB decision that:
- **Interprets Code Provisions**: Clarifies how a specific Code section applies to a project or existing condition
- **Establishes Precedent**: Creates a binding ruling for the subject property
- **Resolves Ambiguity**: Addresses situations where Code language is unclear or multiple interpretations exist
- **Authorizes Non-Compliance** (in some cases): May permit alternative compliance when strict Code compliance is impractical
- **Becomes Part of the Record**: Establishes a documented basis for design, construction, or operational decisions

A Determination is **not** an approval or permit—it is a Code interpretation that informs whether a particular approval or permit can be granted.

## Types of Determinations

### 1. Letter of Determination (LOD)
**Purpose**: Provides a DOB interpretation of Code provisions for a specific project or existing condition.

**When Used**:
- Code language is ambiguous or unclear
- Applicant seeks clarification before filing a major permit application
- A unique site condition warrants special consideration
- Alternative compliance path needs official validation

**Scope**: Addresses a defined question or code issue; does not necessarily authorize the work.

**Timeline**: Typically 30-90 days from complete filing [VERIFY current DOB processing times].

### 2. Appeal Determination
**Purpose**: Reviews and rules on an appeal of a DOB violation, permit denial, or previous DOB decision.

**When Used**:
- A violation has been issued and the property owner disputes it
- A permit application has been denied
- A previous DOB decision is being challenged
- An applicant seeks reversal or modification of a DOB inspector's determination

**Scope**: Limited to the specific issue(s) being appealed.

**Timeline**: Typically 60-180 days; may be longer for complex cases [VERIFY].

### 3. Commissioner's Determination
**Purpose**: A high-level determination issued or endorsed by the DOB Commissioner addressing significant or precedent-setting code interpretation questions.

**When Used**:
- The issue affects multiple properties or establishes citywide precedent
- Previous determinations conflict and need resolution
- The question involves major policy or regulatory implications
- Lower-level determinations have been appealed or contested

**Scope**: May be broader than a standard LOD and can establish wider precedent.

**Timeline**: Extended; typically 90-180+ days [VERIFY].

## When to Pursue a Determination

### Pursue a Determination When:

**1. Code Interpretation is Ambiguous**
- Building Code language is unclear or subject to multiple interpretations
- No prior DOB guidance exists on the specific issue
- The applicant's interpretation differs from DOB's initial position

**2. Unique Site or Project Conditions Exist**
- Property characteristics make standard Code compliance difficult or impossible
- Historic preservation conflicts with modern Code standards
- Structural or site constraints warrant alternative compliance paths
- Existing building conditions cannot practically meet current Code

**3. Precedent is Needed**
- The issue will likely recur on future projects
- A documented ruling supports future permit applications
- Dispute resolution is necessary before committing project resources

**4. Cost or Feasibility Justification**
- Strict compliance is economically impractical
- Demonstrating hardship supports a variation or alternative approach
- A determination can justify design or construction choices to stakeholders

**5. Violation or Denial Review**
- A violation was issued and you dispute the interpretation
- A permit was denied based on DOB's Code reading
- A previous determination conflicts with current DOB requirements

### Do NOT Pursue a Determination When:

**1. Code Language is Clear**
- The Code explicitly requires or prohibits the proposed work
- DOB's position is consistent with the plain language of the Code
- No reasonable alternative interpretation exists

**2. Faster Alternatives Exist**
- A standard permit application will resolve the issue more quickly
- Expedited processing or variance procedures are available
- The determination timeline will cause unacceptable project delays

**3. The Issue is Minor**
- The question affects a small, non-critical building component
- Standard design modifications can achieve compliance
- The cost of a determination exceeds the cost of compliance

**4. Facts are Unclear or Disputed**
- Key site conditions, measurements, or building history are unknown or contested
- You cannot provide definitive documentation to support your position
- Additional investigation is needed before filing

## Determination Filing Process

### Phase 1: Pre-Filing Research and Planning (1-2 weeks)

**Step 1.1: Research Existing Determinations**
- Search the DOB Determination database for prior rulings on the same or similar code provisions
- Review Board of Standards and Appeals (BSA) decisions related to your issue (see BSA Relationship section below)
- Consult DOB policy guidance, guidelines, and Q&A resources
- Document any prior determinations that support your position

**Step 1.2: Clarify Your Legal Question**
- Define the specific Code provision(s) in question
- Document the existing condition or proposed work that creates the ambiguity
- Identify your interpretation and the DOB's position (if known)
- Develop a clear statement of the question you are posing

**Step 1.3: Assess Strategic Value**
- Determine whether a favorable determination is achievable
- Evaluate timeline and cost implications
- Identify alternative paths (variance, appeal, standard permit approach)
- Decide whether to proceed

### Phase 2: Preparation of Determination Application (2-4 weeks)

**Step 2.1: Compile Code Analysis**
Prepare a comprehensive written analysis that includes:
- **Applicable Code Sections**: Cite all Building Code, Administrative Code, and other regulatory provisions relevant to your question
- **Code Language**: Reproduce exact text of the provisions in question
- **Plain Language Summary**: Explain what the Code says in accessible language
- **Your Interpretation**: Present your reading of the Code and why it applies to your situation
- **DOB's Position** (if known): Cite any prior DOB interpretation or rejection of your position
- **Ambiguity or Conflict**: Explain why the Code is unclear or subject to multiple interpretations

**Step 2.2: Gather Supporting Precedent**
- **Prior Determinations**: Include copies of related DOB determinations that support your interpretation
- **BSA Decisions**: Reference Board of Standards and Appeals cases with applicable rulings
- **Code History**: If relevant, explain how the Code provision has evolved and what it was intended to address
- **Industry Standards**: Cite accepted design standards, engineering practices, or professional guidelines that support your interpretation
- **Peer City Examples**: If appropriate, reference how other jurisdictions interpret similar code language

**Step 2.3: Document the Site Condition or Proposed Work**
Assemble documentation that demonstrates why the determination is necessary:
- **Photographs and Site Plans**: Current conditions showing the issue
- **Building Plans**: Existing or proposed floor plans, elevations, and sections
- **Measurements and Specifications**: Accurate dimensions of relevant building elements
- **Historical Documentation** (if applicable): Original plans, prior permits, or records showing the existing condition
- **Technical Reports**: Structural, mechanical, or other engineering analysis supporting your position

**Step 2.4: Prepare Professional Certification**
All determinations require professional certification (seal and signature):
- **Licensed Professional Engineer (PE)** or **Licensed Architect** must review and certify the application
- The professional must attest that the code analysis is accurate and the proposed interpretation is reasonable
- For some determinations, multiple professionals may sign (e.g., structural and electrical engineers)

**Step 2.5: Prepare Written Narrative**
Compose a clear, concise written statement (typically 1-5 pages) that:
- States the question you are asking DOB
- Explains the existing site condition or proposed work
- Summarizes your code interpretation and the supporting analysis
- Requests a specific determination (approval, denial, or interpretation)
- Concludes with a summary statement

### Phase 3: Filing the Determination Application (1 week)

**Step 3.1: Determine Filing Method**
As of the date of this guide, determinations are typically filed through:
- **DOB NOW**: Electronic submission portal [VERIFY if determinations are available through DOB NOW]
- **DOB Customer Service**: In-person or mail submission [VERIFY current procedures]
- **Borough Office**: Filing at the relevant borough DOB office [VERIFY borough-specific procedures]

Confirm the current filing method with DOB before preparing your submission.

**Step 3.2: Complete Application Form**
- Complete the DOB Determination Application form (if available)
- Provide accurate building address, BIN, and project description
- List the specific question(s) being posed
- Identify the primary professional submitting the application
- Verify that all required documents are listed and included

**Step 3.3: Assemble Complete Submission Package**
Organize documents in the following order:
1. Application form and cover letter
2. Statement of question (summary)
3. Code analysis and interpretation memo
4. Supporting documentation (site plans, photos, technical reports)
5. Professional certification and seal
6. Copies of prior determinations or relevant precedent
7. Index or table of contents

**Step 3.4: Submit Application**
- File through the designated method (DOB NOW, in-person, or mail)
- Retain a copy for your records and a signed delivery receipt (if mailing)
- Note the date of submission
- Confirm receipt with DOB if necessary

**Step 3.5: Pay Filing Fee**

### Phase 4: DOB Review and Determination (30-180+ days)

**Step 4.1: Initial Review**
DOB conducts a preliminary review (typically 5-15 days) to confirm completeness. If documents are missing or inadequate, you will receive a Request for Additional Information (RAI). Respond promptly.

**Step 4.2: Technical Review**
DOB's relevant bureau or office reviews the code analysis and site documentation and prepares a determination (typically 15-60 days).

**Step 4.3: Internal Approval**
The determination is reviewed and approved by DOB management (timeline varies by determination type).

**Step 4.4: Issuance**
The Determination is issued in writing and becomes effective. You will receive an official copy via email or mail.

**Step 4.5: Post-Determination Follow-Up**
- Review the determination carefully to confirm it answers your question
- If the determination is unfavorable, evaluate whether to appeal or pursue alternative approaches
- If favorable, use the determination to support subsequent permit applications
- Maintain the determination in your project file for future reference

## Required Documentation

### Essential Documentation for All Determinations:

1. **Determination Application Form** (if required by DOB)
2. **Written Statement of Question**: Clear articulation of the code interpretation question
3. **Code Analysis Memo**: Detailed analysis of applicable code provisions (see Phase 2, Step 2.1)
4. **Building Address and BIN**: Accurate identification
5. **Existing Condition or Project Description**: Clear explanation of the situation requiring the determination
6. **Professional Certification**: Licensed PE or Architect signature and seal

### Typical Supporting Documentation:

- **Site Plans or Building Plans**: Floor plans, elevations, and sections showing the relevant conditions
- **Photographs**: Images of the existing condition or building feature in question
- **Technical Reports**: Engineering analysis, structural reports, mechanical studies
- **Measurements and Specifications**: Precise dimensions relevant to the code question
- **Prior Determinations**: Copies of related DOB determinations
- **BSA Decisions**: References or copies of relevant Board of Standards and Appeals rulings
- **Code Analysis References**: Copies of the specific Code sections cited
- **Project History** (if applicable): Prior permits, violations, or correspondence related to the issue

### Professional Requirements:

- **Licensed Professional Engineer (PE)** or **Licensed Architect**—licensed in New York State
- Professional must review and certify the code analysis
- Professional's seal and signature must appear on all supporting documents
- Professional must attest that the analysis is accurate and reasonable

## Board of Standards and Appeals (BSA) Relationship

### What is the BSA?

The Board of Standards and Appeals is an independent City agency that hears appeals of DOB decisions and grants variances from Building Code requirements.

### DOB Determination vs. BSA Appeal

**DOB Determination**:
- Requests DOB to interpret or clarify the Code
- Asks for DOB's official position on a code question
- Does not request a variance or exception from Code requirements
- May result in DOB confirming that strict Code compliance is required

**BSA Variance or Appeal**:
- Requests the BSA to grant an exception from Code requirements
- Arguments must show unnecessary hardship or that the variance is justified by special circumstances
- Requires proof that the work cannot comply with the Code as written
- Results in a legal variance that allows non-compliant work

### Strategy Considerations

**Use DOB Determination When**:
- You believe the Code supports your interpretation
- You are asking for clarification, not an exception
- You want DOB's official position on the code language
- The issue may recur and you need documented guidance

**Use BSA Variance When**:
- Strict Code compliance is impossible or impractical
- You need an official exception from Code requirements
- You can demonstrate unnecessary hardship or special circumstances
- You are prepared for a hearing and possible public opposition

**Combined Strategy**:
- File a determination first to establish DOB's position
- If unfavorable, then pursue a BSA variance
- Use a favorable determination to support your variance application
- If BSA hearing is required, use the determination in your presentation

## OATH/ECB Determinations vs. DOB Determinations

### OATH (Office of Administrative Trials and Hearings)

**Purpose**: OATH conducts administrative hearings on DOB violations and permit denials.

**When Involved**:
- A violation has been issued and the respondent requests a hearing
- A permit has been denied and the applicant is appealing
- The DOB and applicant disagree on code interpretation

**Proceeding**: A hearing officer reviews evidence and arguments and issues a decision (called an "OATH Decision" or "ECAB Decision" for the Environmental Control Board).

### ECB (Environmental Control Board)

**Purpose**: The ECB is a division of OATH that handles environmental violations, including some DOB violations.

**Relationship to DOB Determinations**:
- OATH/ECB decisions interpret Code provisions, but are limited to the specific violation or permit denial
- A DOB Determination is broader and may address code interpretation questions not yet in dispute
- You can file a Determination before a violation is issued to preempt disputes
- If a violation is issued, an OATH hearing may reference prior Determinations

### Strategic Use

**Pre-Determination Strategy**:
- File a Determination BEFORE a violation to establish DOB's position
- If DOB issues a favorable determination, violations are less likely
- If DOB determines that your interpretation is correct, that becomes precedent

**Post-Violation Response**:
- If a violation is issued after a favorable determination, use the determination as evidence in an OATH hearing
- A determination that conflicts with a violation strengthens your appeal

## Timeline Expectations

### Typical Determination Processing Timeline:

**Simple Letter of Determination** (narrow code question, straightforward facts):
- **Preparation**: 2-4 weeks
- **DOB Processing**: 30-60 days
- **Total Time**: 6-12 weeks

**Complex Letter of Determination** (broader question, multiple code provisions, difficult facts):
- **Preparation**: 4-8 weeks
- **DOB Processing**: 60-120 days
- **Total Time**: 3-5 months

**Appeal Determination** (response to violation or permit denial):
- **Preparation**: 2-4 weeks
- **DOB Processing**: 60-180 days
- **Total Time**: 2-6 months

**Commissioner's Determination** (high-level, precedent-setting):
- **Preparation**: 4-8 weeks
- **DOB Processing**: 120-180+ days
- **Total Time**: 4-8 months or longer

### Timeline Considerations:

- DOB processing times vary based on workload and determination complexity
- Incomplete submissions or RAIs add 2-4 weeks
- Expedited processing may be available [VERIFY if DOB offers expedited determination service]
- Summer and year-end periods may see slower processing
- Complex determinations requiring Commissioner review take significantly longer

**Planning Note**: Account for determination timeline in your overall project schedule. If the timeline is unacceptable, consider alternative paths (standard permit, BSA variance, or design modifications).

## Common Scenarios Where Determinations Are Needed

### Scenario 1: Existing Building Cannot Meet Current Code
**Situation**: A historic building or existing structure cannot comply with current Building Code provisions (headroom, ceiling height, window sizing, egress width).

**Determination Question**: "Does the Code requirement for [specific provision] apply to existing buildings? If so, what alternatives exist?"

**Outcome**: Determination clarifies whether existing buildings are exempt or whether alternative compliance exists.

### Scenario 2: Code Language is Ambiguous
**Situation**: The Code says a certain element is required "as specified in Section X," but Section X has multiple possible interpretations.

**Determination Question**: "In applying Section X to [specific building condition], which interpretation is correct?"

**Outcome**: DOB clarifies which interpretation applies.

### Scenario 3: Unique Site Condition
**Situation**: A narrow lot, unusual shape, or site constraint makes standard Code compliance impossible.

**Determination Question**: "Given [specific site constraint], what alternative approach satisfies the Code's intent for [code provision]?"

**Outcome**: Determination authorizes an alternative design approach.

### Scenario 4: Technology or Materials Not Covered in Code
**Situation**: A new building material, mechanical system, or technology is not explicitly addressed in the Building Code.

**Determination Question**: "Is [new material/system] equivalent to or acceptable as an alternative to [code-specified material/system]?"

**Outcome**: Determination authorizes use of the new material or system if deemed equivalent.

### Scenario 5: Conflict Between Code Sections
**Situation**: Two different Code provisions appear to conflict or require incompatible designs.

**Determination Question**: "When Code sections X and Y conflict, which provision controls in [specific situation]?"

**Outcome**: Determination resolves the conflict.

### Scenario 6: Prior DOB Position is Unclear
**Situation**: You have received conflicting guidance from different DOB inspectors or divisions on a code question.

**Determination Question**: "What is the DOB's official interpretation of [code provision] as it applies to [your situation]?"

**Outcome**: Determination establishes official DOB position and prevents future disputes.

## How to Research Existing Determinations

### Step 1: Access the DOB Determination Database
- [VERIFY: Current location of DOB determination database/records]

### Step 2: Search by Code Section
Search for determinations that reference the specific Building Code section you are researching.

### Step 3: Search by Building Type or Use
Filter by building type (residential, commercial, industrial) to find relevant prior determinations.

### Step 4: Search by Topic
Use keywords related to your code question (e.g., "headroom," "alteration," "equipment," "egress").

### Step 5: Review and Analyze
- Read the prior determination carefully to understand the question and DOB's reasoning
- Note the date and any changes to the Code since the determination
- Assess whether the prior determination supports or contradicts your position
- Document the citation for use in your new determination application

### Step 6: Request Determination Copies
If you cannot find a specific determination in the database:
- Contact DOB Customer Service to request a copy
- Contact the relevant borough DOB office
- Submit a FOIL (Freedom of Information Law) request if necessary
