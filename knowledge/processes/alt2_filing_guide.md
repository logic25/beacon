# ALT-2 (Alteration Type 2) Filing Guide

## Overview

ALT-2 is the filing designation for **minor alterations** involving **multiple work types (trades)** that do not trigger major alteration thresholds. ALT-2 is faster than ALT-1 and more flexible than single-trade PAA filings, making it the workhorse filing for typical renovation and upgrade projects.

### What Qualifies as ALT-2

ALT-2 is appropriate when:

- **Multiple work types are involved** — Combination of plumbing, electrical, mechanical, and/or construction trades
- **No occupancy change** — Building use/occupancy classification remains the same
- **No new egress modifications** — No changes to exit routes, stairwells, or travel distances
- **No structural changes to primary system** — No modifications to columns, primary beams, load-bearing walls, or floor systems
- **No legalization work** — Not creating a formal legal record of non-compliant conditions

**Common ALT-2 Project Types:**
- Interior renovations (kitchen, bathroom, office reconfiguration)
- HVAC system replacement
- Plumbing or electrical upgrades/riser modifications
- Tenant fit-outs in commercial space
- Facade work (with structural engineering but not primary system changes)
- MEP replacements in occupied buildings
- Mixed-trade apartment renovation

---

## ALT-2 vs. PAA vs. ALT-1: Filing Decision Guide

| Criterion | ALT-2 | PAA | ALT-1 |
|-----------|-------|-----|-------|
| Multiple trades? | **Yes** | Single (assembly-specific) | Yes |
| Change of occupancy? | No | Special (assembly use) | **Yes** |
| New egress? | No | Possible | **Yes** |
| Structural changes? | No (non-primary) | No | **Yes** |
| Timeline | Fast (1–3 weeks approval) | Moderate | Slow (4–8+ weeks) |

**Quick Rule:**
- Multiple trades, no occupancy change → **ALT-2**
- Single work type (electrical only, plumbing only) → **PAA** (single trade)
- Occupancy change, new egress, or structural work → **ALT-1**

## When ALT-2 vs. Single-Trade PAA

**ALT-2:** Multiple trades (e.g., plumbing + electrical + HVAC)
**PAA (Single-Trade Permit):** Only one work type (e.g., electrical only, plumbing only)

If your project involves electricians AND plumbers, file ALT-2. If only one trade, file the applicable single-trade permit.

## Key ALT-2 Distinction: What You CANNOT Include

Do not file ALT-2 if your project involves:

- **Occupancy change** (use change) → Use ALT-1
- **New egress routes** (stair relocation, new exits) → Use ALT-1
- **Primary structural changes** (column removal, beam upsizing, load-bearing wall modification) → Use ALT-1
- **Legalization of existing non-compliant conditions** → Use ALT-1 (no-work)
- **Combination of structural work + egress changes** → Use ALT-1

If you mistakenly file ALT-2 when ALT-1 is required, DOB will reject and reassign the application, causing delays.

## Common ALT-2 Scenarios

### Scenario 1: Interior Renovation (Residential Apartment)
Kitchen + bathroom + flooring update in a rent-stabilized building.

**Why ALT-2?** Multiple trades (plumbing, electrical, carpentry); no occupancy change.

**Typical scope:** Plumbing risers upgraded, electrical panel circuit additions, new lighting, drywall/painting.

### Scenario 2: Commercial Tenant Fit-Out
Office space reconfiguration for new tenant; includes partitions, mechanical distribution, electrical outlets.

**Why ALT-2?** Multiple trades; occupancy class remains "office"; no egress changes (existing exits adequate).

**Note:** Verify occupant load does not increase beyond existing egress capacity.

### Scenario 3: HVAC System Replacement
Building-wide or floor-level heating/cooling system swap with new ductwork and controls.

**Why ALT-2?** Single primary trade (mechanical) but may involve electrical connections and minor structural supports; no occupancy impact.

### Scenario 4: Plumbing Riser Upgrade
Adding new water/waste lines to serve additional fixtures or modernize old piping.

**Why ALT-2?** Plumbing + minor carpentry (wall openings); no occupancy change.

### Scenario 5: Facade Work with Non-Structural Modifications
Facade inspection (FISP) repairs that include non-load-bearing masonry and caulking, with some minor electrical (lighting).

**Why ALT-2?** Multiple trades; facade work does not affect primary structure.

## Plan Examination: Professional Certification vs. DOB Review

### Professional Certification (Pro-Cert) Track
**Applicant chooses:** Have a licensed PE/RA professionally certify the plans instead of submitting to DOB examination.

**Advantages:**
- Faster approval (certification sign-off vs. examiner review)
- Plans do not go through DOB queue
- Flexibility in design iteration before final certification

### DOB Review Track
**Applicant chooses:** Submit plans for traditional DOB examination.

- DOB takes responsibility for plan review
- Lower risk of post-permit audit corrections
- Clearer approvals before permit issued

**Disadvantages:**
- Longer review timeline (2–4 weeks typical)

- Queue delays

## Cost Affidavit and Job Estimate

The cost affidavit declares the **total estimated construction cost** of work. This affects:
- Filing fee calculation
- Inspection scope
- Potential expedite fee eligibility

**Accuracy matters:** Significantly underestimating cost can trigger DOB objections or post-permit audits. Overestimating may increase fees.

Obtain signed affidavit from contractor or architect before filing.

## Common ALT-2 Mistakes (and How to Avoid Them)

| Mistake | Consequence | Prevention |
|---------|-------------|-----------|
| File ALT-2 when ALT-1 required (occupancy change) | Application reassigned/rejected; major delays | Verify occupancy does NOT change; confirm with GLE before filing |
| Missing work types in scope | Inspections incomplete; work stoppage mid-project | List all trades involved upfront; coordinate with contractor |
| Underestimated cost affidavit | DOB questions; potential audit post-permit | Use detailed contractor estimate; include contingency |
| Forgot to seal plans | Plan rejection; resubmit with PE/RA seal | Coordinate PE/RA sign-off before file submission |
| No FDNY coordination for mechanical/sprinkler work | Mechanical/sprinkler objections after approval | Verify if work triggers FDNY filing; coordinate early |
| Missing TPP for occupied building alterations | Tenant complaint; potential violation issued | Include TPP if dwelling units occupied during construction |

## When ALT-2 Becomes ALT-1

If during plan development or construction, the scope expands to include:
- Occupancy change
- New egress modifications
- Primary structural modifications

**Stop and convert to ALT-1.** File an ALT-1 amendment or close the ALT-2 and open new ALT-1 filing.

## Final Inspection and Closeout

### Final Inspection Request
Submit Final Inspection request in Build once construction complete. Inspector verifies:
- Work matches approved plans
- All required inspections passed
- No outstanding violations

### Completion Certificate
Upon Final Inspection approval, DOB issues completion documentation (Letter of Completion, not Certificate of Occupancy, since occupancy unchanged).
