# Means of Egress Requirements — NYC Building Code Guide

## Overview

**Means of egress** = the continuous and unobstructed path of travel from any occupied point in a building to a public way (street, plaza, etc.). It is the foundation of life safety code and one of the most frequently cited DOB violations.

The NYC Building Code Chapter 10 (2022 edition) governs egress requirements. Understanding occupant load calculations, exit quantity, stairway sizing, and travel distance is essential for any expediting project involving alterations, change of use, or new construction.

**Critical:** Egress violations are common DOB findings. Early code analysis prevents objections and delays.

---

## Key Concepts and Terminology

### Occupant Load

**Definition:** The maximum number of persons the building, floor, or room can accommodate based on use and floor area.

**Calculation:**
- Determined by dividing floor area by **occupant load factor** (varies by use)
- Examples:
  - Residential: 200 SF per occupant (one person per 200 SF)
  - Office: 250 SF per occupant
  - Retail/Mercantile: 60 SF per occupant
  - Assembly (theater): 7 SF per occupant (most stringent)

**Purpose:**
- Determines number of exits required
- Determines egress width (stairways, doors, corridors)
- Determines travel distance allowance

**Note:** Occupant load calculated at **most restrictive use** on a floor. A mixed-use floor gets the most stringent occupant load factor.

### Exit Access

**Definition:** The portion of the egress path from any occupied space to an exit.

**Components:**
- Rooms/office spaces
- Corridors
- Aisles in larger spaces
- Doors to exits

**Requirements:**
- Must be unobstructed, clearly marked
- Illuminated and marked with exit signage
- Maximum travel distance limits apply (varies by occupancy)

### Exit

**Definition:** Protected means of egress (enclosed stairway, exterior door at grade, horizontal exit, ramp).

**Types:**
- **Enclosed stairway** (most common) — fire-rated enclosure with self-closing doors
- **Exterior exit stairway** (outside the building)
- **Horizontal exit** (passage to adjacent building or refuge area)
- **Direct exit to public way** (door at grade level)
- **Ramp** (accessible exit)

**Critical:** Exit must be protected from fire/smoke. Direct occupancy (offices opening to stairway) NOT permitted on exit stairs in many cases.

### Exit Discharge

**Definition:** Portion of egress path from the exit to a public way.

**Components:**
- Lobby (if between exit and public way)
- Exterior path to street/sidewalk
- Requirements vary by occupancy

## Number of Exits Required

Exit quantity is determined by occupant load:

| Occupant Load | Minimum Exits |
|--------------|--------------|
| 1-49 occupants | 1 exit (with conditions) |
| 50-500 occupants | 2 exits |
| 501-1,000 occupants | 3 exits |
| Over 1,000 occupants | 4 exits |

### Important Clarifications

**Single Exit (49 or fewer occupants):**
- Permitted only in certain occupancy groups (offices, retail, etc.)
- NOT permitted in assembly, residential, educational, institutional occupancies
- Additional code requirements apply (travel distance, common path limits)

**Most NYC buildings require minimum 2 exits** due to occupancy load, height thresholds, and occupancy type restrictions.

**High-rise buildings (75+ feet):** Additional exit requirements may apply; typically 2+ exits per floor minimum.

## Egress Width Requirements

### Stairway Width (Most Common Issue)

**Calculation:**
- **With sprinklers:** 0.3 inches per occupant (maximum)
- **Without sprinklers (certain occupancies):** 0.2 inches per occupant

**Example:**
- Floor with 300 occupants (without sprinklers)
- Stairway width = 300 × 0.2 = 60 inches minimum (two 30-inch doors or one 60-inch opening to stairwell)

**Minimum widths:**
- 44 inches — standard minimum for general occupancies
- 36 inches — only for floors with occupant load under 50
- 48 inches — assembly occupancies and certain high-occupancy uses

**Common Issue:** Existing buildings with 36-inch stairways may not meet code for current occupancy. Widening stairs required, which is costly and disruptive.

### Door Width (Exit and Exit Access)

- **Minimum clear opening:** 32 inches (measured from face of door to edge of door frame when open)
- Doors must swing in direction of travel (panic hardware required for assembly occupancies)
- Double doors must have pair width meeting width calculation

### Corridor Width

- **Standard minimum:** 44 inches
- **Reduced (low occupancy):** 36 inches
- **Assembly occupancies:** Wider requirements apply

## Travel Distance

**Definition:** Maximum distance occupant must walk from any point to reach an exit or exit access.

### Travel Distance Limits

| Occupancy | With Sprinklers | Without Sprinklers |
|-----------|-----------------|-------------------|
| General (B, M, S) | 250-300 feet | 150-200 feet |
| Residential (R) | 125 feet | 75 feet |
| Assembly (A) | 200-250 feet | 150 feet |
| Educational (E) | 250 feet | 150 feet |
| Hazardous (H) | 50-75 feet | 25-50 feet |

**Measurement:** Measured along the actual path of travel, not straight-line distance.

### Dead-End Corridors

**Definition:** A corridor or path with only one way out.

**Limits:**
- **General:** Maximum 20-feet dead-end (occupancy dependent)
- **With sprinklers:** May extend to 50 feet in certain cases
- **Existing buildings:** More lenient limits may apply [VERIFY: Exact limits for existing buildings?]

**Common Issue:** Existing floor layouts with dead-end hallways to restrooms or small offices. Solutions: Add exit door, reduce dead-end length, or remove obstruction.

## Fire Escapes in Egress Design

### New Construction

- **Fire escapes NOT acceptable** as primary means of egress
- Cannot be counted toward exit quantity or width requirements
- May be supplementary egress in rare circumstances with code official approval

### Existing Buildings

- **May be acceptable as ONE of the two required exits** in certain existing residential buildings
- **MDL §53** (Multiple Dwelling Law) — governs fire escape requirements for multiple dwellings
- **1 RCNY §15-10** — governs fire escape maintenance, testing, compliance
- Fire escapes must be maintained in code-compliant condition

## Common Egress Issues GLE Encounters

### Insufficient Number of Exits

**Problem:** Floor has 200 occupants but only 1 exit (old commercial space now being re-occupied).

**Solution:**
- Add second exit (new stairway, exterior door at grade, or horizontal exit)
- OR reduce occupancy through design (zone occupancy, density reduction)
- OR demonstrate that building code in effect when built applies (existing building grandfather clause)

**Impact:** Significant cost and timeline; requires structural design work.

### Egress Width Too Narrow

**Problem:** Existing 36-inch stairway now serving 300 occupants; code requires 60+ inches.

**Solution:**
- Widen stairway (structural modification, construction impact)
- OR reduce occupancy to match existing width
- OR add sprinkler system (reduces width requirement)

**Note:** Widening stairs in existing buildings is very expensive due to structural changes.

### Dead-End Corridor Exceeding Limit

**Problem:** Hallway to restroom is 35 feet; code limit is 20 feet.

**Solutions:**
- Add exit door at end (if feasible)
- Reduce occupancy load to corridor
- Install sprinkler system (may extend allowance)
- Remove obstruction/obstacles making corridor shorter in practical path

### Exit Through Another Tenant's Space

**Problem:** Unit's only exit passes through adjacent tenant's space.

**Solution:** NOT PERMITTED under any circumstances
- Second exit must be direct to public way or independent
- This is a common violation in reconfigured office floors
- Requires design change (new stairway, separate egress path)

### Stairway Enclosure and Fire Rating

**Issue:** Stairway must be enclosed and fire-rated.

**Requirements:**
- Enclosure rated (typically 1-2 hour, depending on building height and occupancy)
- Self-closing, automatic-closing doors at each floor
- No direct office occupancy on stairway landings
- Proper ventilation (smoke control)

**Common Problem:** Existing stairways may not be properly enclosed/rated; enclosure upgrades required for ALT II.

## Occupancy Group Specific Requirements

### Group A (Assembly)

**Most Stringent Egress Requirements**

- Theater seats: Specific aisle widths (36-44 inches depending on row length)
- Clear egress paths; no obstacles
- Panic hardware required on doors
- Exit signage must be visible from all seats
- Dead-end corridor limits: Very strict (10-20 feet)
- Emergency lighting and illuminated exit signage mandatory

**Examples:** Theaters, nightclubs, restaurants with performance areas, conference halls

### Group B (Business)

**Standard Requirements**

- Typical occupant load: 250 SF per occupant
- Standard exit and travel distance requirements apply
- Offices configured for 50+ occupants: 2 exits minimum
- Corridors: 44 inches minimum

**Examples:** Office buildings, call centers, banks, laboratories

### Group R (Residential)

**Stringent Requirements**

- Occupant load: 200 SF per occupant
- **Two exits from each dwelling unit** (or two separate egress paths)
- Common corridor must have 2 exits
- Exit access from unit to common corridor, then to exterior exits
- Higher fire rating for corridors in apartments
- Single-stair buildings may not be permitted above certain heights

**Examples:** Apartments, hotels, dorms, nursing homes

### Group M (Mercantile)

**Retail-Specific Requirements**

- Occupant load: 60 SF per occupant (higher density than office)
- Aisle width requirements (minimum 24-36 inches)
- Clear paths to exits from shopping floor
- Stockroom occupancy separate from retail floor

**Examples:** Retail stores, supermarkets, shopping centers

## Existing Buildings and the 2022 NYC Building Code

### Compliance Triggers for Existing Buildings

**Existing buildings generally comply with the code in effect when they were built.** However, code compliance is required when:

- **Substantial alteration** occurs (triggering compliance for altered systems/areas)
- **Change of occupancy** (different use may require egress reassessment)
- **Specific trigger points** for life safety upgrades

## Pro Tips and Common Solutions

[VERIFY: What are the most common egress objections your team encounters? What design solutions or phasing strategies work best?]

### Early Code Analysis

- Conduct occupant load and egress analysis in design phase
- Identify deficiencies before DOB submission
- Reduces objections and redesign cycles

### Occupancy Load Reduction

- If egress cannot be added, reduce occupancy (remove office seats, reduce retail area, etc.)
- Lower occupancy → fewer exits required, smaller widths acceptable
- Often more cost-effective than structural egress work

### Sprinkler System Value

- Installing sprinkler system can reduce egress width requirements (0.3 vs. 0.2 inches per occupant)
- May extend travel distance allowances
- Often justifiable for mixed-use or alteration projects

### Grandfather Clause vs. Current Code

- Consult code official on whether building qualifies for existing code grandfathering
- Substantial alteration may trigger current code; minor alterations may not
- Early DOB pre-submission meeting clarifies expectations

### Phasing Strategy

- Phase alterations to avoid triggering building-wide code compliance
- Obtain ALT I approval for Phase 1 before Phase 2 scope expands
- Negotiate with DOB on scope definitions

### Design Solutions for Tight Spaces

- Horizontal exits to adjacent buildings (if available)
- Outdoor stair enclosures (weather-resistant, fire-rated)
- Ramps as secondary exit (accessible egress)
- Vestibule/lobby reconfigurations to achieve width

## Key Takeaways

1. **Occupant load is the foundation**: Accurate calculation drives all egress requirements
2. **Two exits are common minimum**: Most NYC projects require minimum 2 exits per floor
3. **Travel distance and width are often constraints**: Particularly in existing buildings with fixed layouts
4. **Dead-end corridors are frequent violations**: Address during design to avoid DOB objections
5. **Sprinkler systems have egress benefits**: Consider for mixed-use or alteration projects
6. **Fire escapes have limited role**: Acceptable only in existing residential under specific circumstances
7. **Early code analysis saves costs**: DOB objections on egress late in design are expensive to resolve
8. **Existing buildings may have relief**: Grandfather clause or alternative compliance options available if analyzed early
