# When to File PAA vs ALT Filings - Decision Tree

**IMPORTANT UPDATE:** As of DOB NOW Build implementation, the filing nomenclature has changed:
- **OLD:** ALT1, ALT2, ALT3
- **NEW:** Alt-CO (Alteration - Certificate of Occupancy) and Alterations

**This guide covers BOTH systems:**
- Current filings use Alt-CO and Alterations
- Legacy ALT1/ALT2/ALT3 references apply only to closing out old applications

**Critical clarification:** PAA = **Post-Approval Amendment**, NOT Place of Assembly. Place of Assembly is abbreviated as "PA" (different thing entirely).

---

## Quick Reference Decision Tree (DOB NOW Build)

```
START: Do you have an APPROVED permit already?
│
├─ YES → Is the change MINOR (doesn't affect structure, egress, occupancy)?
│   ├─ YES → File PAA (Post-Approval Amendment)
│   └─ NO → File new Alterations application (may need to withdraw original)
│
└─ NO (no approved permit yet) → What type of work are you doing?
    │
    ├─ ONLY changing use/occupancy, NO construction → Alt-CO
    │
    └─ Construction work (any scope) → Alterations
```

**Note:** The old ALT1/ALT2/ALT3 distinction is gone. DOB NOW Build simplifies to:
- **Alt-CO** = Occupancy/use changes only (replaces ALT1)
- **Alterations** = Any construction work (replaces ALT2 and ALT3)

---

## PAA (Post-Approval Amendment)

### What It Is:
A **Post-Approval Amendment** allows you to make changes to plans AFTER a permit has been approved but BEFORE work is complete.

### When to Use PAA:

**Use PAA if ALL of these are true:**
1. ✅ You have an **approved permit** (ALT or NB job number exists and is approved)
2. ✅ The change is **minor** and doesn't fundamentally alter the approved scope
3. ✅ The change doesn't affect:
   - Structural elements in a significant way
   - Egress/fire safety provisions
   - Occupancy group or use
   - Building envelope (significantly)

### Common PAA Scenarios:
- Relocating interior partitions (non-structural)
- Minor plumbing fixture changes
- Adjusting window sizes (within approved envelope)
- Material substitutions (e.g., different tile, fixtures)
- Minor MEP routing changes

### PAA Process:
1. Submit amended plans via DOB NOW Build
2. Reference original approved job number
3. Highlight changes from approved plans
4. Pay PAA filing fee (typically lower than new filing)
5. Approval timeline: 2-4 weeks (faster than new ALT filing)

### Important PAA Rules:
- ❌ **Cannot use PAA to add scope** beyond what was approved
- ❌ **Cannot change occupancy** or use group via PAA
- ❌ **Cannot add structural work** that wasn't in original scope
- ✅ **Can use multiple PAAs** on the same job if needed
- ✅ **Approved PAA becomes part of approved plans** for TCO/CO

## Alt-CO (Alteration - Certificate of Occupancy)
**Formerly known as: ALT1**

### What It Is:
Changes to **use or occupancy ONLY** with **NO construction work**.

### When to Use Alt-CO:

**Use Alt-CO if:**
1. ✅ You're **changing the use** of a space (e.g., retail to restaurant)
2. ✅ You're **changing occupancy group** (e.g., B to A-2)
3. ✅ There is **NO construction work** involved (no demo, no build-out)
4. ✅ You just need to **legalize a use change** or update Certificate of Occupancy

### Common Alt-CO Scenarios:
- Converting office (B) to medical office (still B, but different use)
- Changing restaurant type (from full service to fast food)
- Adding occupancy to existing space (residential to live/work)
- Legalizing an existing unpermitted use (no work, just paperwork)

### Alt-CO Process:
- Simplest alteration filing type
- Minimal plan requirements (floor plans, occupancy load calculations)
- Approval timeline: 2-4 weeks typically
- No inspections required (usually)

### Alt-CO Does NOT Require:
- ❌ Construction drawings
- ❌ Structural analysis
- ❌ MEP if no systems changes
- ❌ DOB inspections (in most cases)

## Alterations (Construction Work)
**Formerly known as: ALT2 and ALT3**

### What It Is:
Any physical construction work, regardless of scope. DOB NOW Build eliminated the ALT2/ALT3 distinction.

### When to Use Alterations:

**Use Alterations if:**
1. ✅ You're doing **any construction/renovation work**
2. ✅ Work includes:
   - Interior renovations
   - Façade work
   - MEP upgrades
   - Structural repairs/modifications
   - Accessibility improvements
   - **Adding square footage** (formerly ALT3)
   - **Increasing building height** (formerly ALT3)

### Scope Specification:
While DOB NOW Build doesn't distinguish ALT2 vs ALT3 in filing type, you still must specify:
- **Scope of work** in the application
- **Whether you're increasing FAR or height**
- **Zoning compliance** (especially if adding area)

### Common Alterations Scenarios:
- Tenant fit-out / interior build-out (formerly ALT2)
- Kitchen/bathroom renovations (formerly ALT2)
- Façade work (formerly ALT2)
- Rooftop addition (formerly ALT3)
- Horizontal expansion (formerly ALT3)
- Cellar-to-basement conversion (formerly ALT3)

### Alterations Typical Requirements:
- Architectural plans
- Structural plans (if affecting structure)
- MEP plans (if changing systems)
- Fire protection plans (if sprinkler/alarm work)
- Zoning analysis (to confirm compliance, especially if adding FAR)

### Alterations Approval Timeline:
- Plan exam: 4-12 weeks (depending on complexity)
- Objections common: structural calcs, egress compliance, accessibility
- Inspections required: progress, final

### Key Consideration for Large-Scope Work:
Even though DOB NOW Build doesn't distinguish filing types, work that **adds FAR or height** still requires:
- Zoning compliance verification
- Full structural analysis
- Potential building-wide upgrades (elevator, sprinkler, accessibility)
- Longer approval timeline

## Special Case: When to File Multiple Filings

### Scenario: Renovation + Use Change

**Example:** Converting retail (M) to restaurant (A-2) with full interior build-out.

**Option 1: ALT2 Only**
- File ALT2 with construction work AND use change noted
- Single filing, single approval process
- **Recommended for most projects**

**Option 2: ALT1 + ALT2**
- File ALT1 for use change first (get approved quickly)
- Then file ALT2 for construction work
- More complex, usually unnecessary
- **Only use if:** You need the use change approved ASAP for business reasons

## Special Case: PAA vs New ALT Filing

### When to Abandon Original and File New ALT:

**File NEW ALT instead of PAA if:**
1. ❌ The changes are **too significant** to qualify as PAA
2. ❌ The original scope was **incorrectly approved** and needs major revision
3. ❌ The original filing is **stale** (approved years ago, never built)
4. ❌ You're essentially **starting over** with a different design

**Example:** Original ALT2 approved for office build-out. Client now wants restaurant with full kitchen, grease trap, hood, sprinkler upgrades. 
→ **File new ALT2**, don't try to PAA this level of change.

## Common Mistakes (What NOT to Do)

### ❌ Mistake #1: Using PAA When You Should File New ALT
**Problem:** Trying to PAA major structural changes or occupancy changes.  
**Result:** DOB rejects PAA, delays project, have to file new ALT anyway.  
**Fix:** If in doubt, ask: "Would an examiner consider this a minor change?" If no, file new ALT.

### ❌ Mistake #2: Filing ALT3 When ALT2 Works
**Problem:** Client is adding a mezzanine but it doesn't increase total building FAR (existing FAR was under-utilized).  
**Result:** Unnecessarily complex filing, longer review, higher fees.  
**Fix:** Calculate FAR first. If mezzanine doesn't exceed existing legal FAR, file ALT2.

### ❌ Mistake #3: Filing ALT1 for Work That Requires Construction
**Problem:** Client wants to "just change the use" but that requires egress improvements, accessibility upgrades, etc.  
**Result:** DOB rejects ALT1, you have to re-file as ALT2.  
**Fix:** Always ask: "Does this use change trigger any construction work?" If yes, file ALT2.

## GLE Pro Tips

### Tip #1: PAA Timeline Advantage
If you have an approved permit and changes are minor, **PAA is almost always faster and cheaper** than filing a new ALT. Use it when you can.

### Tip #2: Coordinate FDNY Early
For ALT2/ALT3 projects involving fire alarm or sprinkler work, **file FDNY simultaneously with DOB** to avoid delays. Don't wait for DOB approval to start FDNY process.

### Tip #3: Pre-File Zoning Check
For ALT3 projects, **run zoning analysis BEFORE filing**. Confirm you have available FAR. If not, you may need zoning variance or need to reduce scope.

### Tip #4: When in Doubt, Call DOB
If you're genuinely unsure whether something qualifies for PAA vs new ALT, **call the borough plan exam unit** and describe the scope. They'll tell you. Better to ask upfront than get rejected later.

## Decision Matrix (Quick Reference)

| Scenario | Filing Type | Why |
|----------|------------|-----|
| Approved permit, minor plan change | PAA | Fastest, cheapest |
| Use change only, no construction | ALT1 | No construction drawings needed |
| Interior renovation, no FAR increase | ALT2 | Standard renovation filing |
| Façade work, no addition | ALT2 | Not increasing building size |
| Adding square footage to building | ALT3 | Increasing FAR |
| Adding floors or rooftop addition | ALT3 | Increasing height/FAR |
| Converting retail to restaurant with build-out | ALT2 | Construction + use change = ALT2 |
| Legalizing existing space (no work) | ALT1 | Paperwork only |
