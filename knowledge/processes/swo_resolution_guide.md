# Stop Work Order (SWO) Resolution Guide

## What Is a Stop Work Order?

A **Stop Work Order (SWO)** is an official DOB directive to cease all (or specific) construction activity at a site. The SWO is:
- Posted physically at the job site on the building or fence
- Issued by DOB inspectors when violations warrant work cessation
- Legally binding; violation is a **criminal offense**
- One of the most urgent issues an expediter must address

**Critical**: Work cannot resume until SWO is rescinded by DOB. Continued work violates the SWO and constitutes a misdemeanor.

## Types of Stop Work Orders

### Full SWO
- All construction work must cease immediately
- No work of any kind permitted on site
- Entire project is halted

### Partial SWO
- Only specific work types are prohibited
- Other approved work can continue
- Example: "Structural work only" — means only structural work stops; MEP rough-ins can continue if separately permitted

**Strategy**: Request partial SWO if only one scope item is problematic; allows project to continue partially while resolving specific issue.

## Common Reasons for SWO

| Reason | Severity | Typical Fix |
|--------|----------|-------------|
| **Work without permit** | High | File PA application (PAA, ALT-1, ALT-2) |
| **Work contrary to approved plans** | High | Amend PA, re-design, or file alteration |
| **Unsafe conditions** | Critical | Engineer certification; structural repair |
| **Failure to maintain site safety** | High | Safety plan update; contractor certification |
| **Pro-cert revoked** [VERIFY: Real example: 170 7th Ave South] | Critical | May require withdrawal and refiling of audited PA |
| **Expired permit** | Medium | Renew permit or file new application |

| **Failure to post required certifications** | Low-Medium | Post required certs; notify DOB |
| **Unpaid DOB violations** | Medium | Pay default penalties |

## Resolution Process: Step-by-Step

### Step 1: STOP ALL WORK IMMEDIATELY
- Notify job superintendent and construction crew
- Post notice at site
- Secure work area
- Document the work stoppage date/time
- **Do not attempt any remedial work until SWO cause is identified**

### Step 2: Identify the SWO Cause
Check multiple sources:
- **BIS violation lookup** (a810-bisweb.nyc.gov): Search address for recent violation
- **DOB NOW**: Check permit file for inspector notes
- **Physical SWO notice**: Read exact violation description on posted notice
- **Contact DOB 311 or inspector**: Call DOB to clarify reason

**Document**: Save violation number, class, description, and date issued.

### Step 3: Correct the Underlying Condition

**If Work Without Permit:**
- Determine scope of unpermitted work
- File appropriate application:
  - **PAA** (Property Alteration Application): Minor alterations
  - **ALT-1**: Alteration to existing building
  - **ALT-2**: New building
- Work must pause until permit issued
- May require design modification to match approved plans

**If Pro-Cert Revoked:**
- Identify why pro-cert was audited and revoked
- Determine if issue can be corrected
- [VERIFY: Must you always withdraw the audited PA and refile? Or can you submit amended cert?]
- Refile with corrected pro-certification
- May be slow process; work remains stopped during resubmittal

**If Unsafe Conditions:**
- Hire licensed professional engineer (PE)
- Engineer inspects condition
- Engineer certifies corrective work needed
- Contractor performs repairs per engineer spec
- Engineer certifies correction complete
- Submit engineer certification to DOB

**If Expired Permit:**
- Renew permit (if project still active within permit term)
- Or file new application if permit expired without renewal

**If Insurance Issue:**
- Obtain proof of required coverage
- Submit to DOB with cover letter

### Step 4: Pay Associated Civil Penalties
SWO violations typically include civil penalties:
- Violations attached to SWO are paid via OATH/ECB process
- Penalties must be paid (or payment arrangement made) before SWO rescission
- Use Certificate of Correction process if violation can be corrected

### Step 5: Request SWO Rescission
Submit to DOB:
- **Letter requesting rescission**
  - Reference violation number and SWO date
  - Describe corrective action taken
  - Attach supporting documentation
  - Request DOB inspection to verify corrections
- **Documentation package**:
  - Engineer certifications
  - Permits for remedial work
  - Proof of insurance
  - Photos of corrected conditions
  - Certificate of Correction (if applicable)
- **Submit via**: DOB NOW or hand-delivered to relevant DOB office

### Step 6: DOB Inspection
- DOB inspector visits site
- Inspector verifies corrections match violation description
- Inspector verifies work is compliant with approved plans
- Inspector documents compliance

### Step 7: SWO Lifted
Upon successful inspection:
- SWO is rescinded (removed from system)
- Violation marked "corrected" or resolved
- **Work can resume** immediately

## Impact on Property and Transaction

**Property record consequences**:
- SWO visible on BIS property record
- May appear on title search (for lenders, purchasers)
- Can affect property valuation

**Closings/refinancing**:
- Lenders may require SWO resolution before closing
- Purchasers may demand SWO cleared as condition of sale
- Title insurance may be withheld pending resolution

**Insurance**:
- Liability/property insurers may deny claims for work under SWO
- Bonding may be affected if work continues under SWO

## Criminal Liability: The Serious Risk

**Violating a SWO is a criminal offense**:
- **Misdemeanor**: Continuing work with active SWO
- **Penalties**: Fines up to $25,000 and/or jail time
- **Who is liable**: Contractor, construction manager, project manager, owner if aware

**Liability extends to**:
- On-site supervisors directing work
- Workers continuing under direction
- Project owners/developers if work continues with knowledge
- Not just the permit holder

**Practice**: Once SWO is posted, work MUST stop. The cost of 1-2 weeks delay is far less than criminal exposure.

## Partial SWO Strategy

If violations affect only part of project scope:
1. **Identify non-affected work**: What can legally continue?
2. **Request modification**: Ask DOB if SWO can be narrowed to specific scope
3. **Document approval**: Get written confirmation before resuming
4. **Separate permits**: Ensure non-SWO work is separately permitted and authorized

**Example**: If structural work is SWO'd for being without permit, but MEP rough-in was separately permitted, MEP work can continue if clearly separated and documented.

## Pro Tips from GLE Practice

- **Speed matters**: Every day stopped is lost productivity; prioritize rescission
- **Engineer involvement**: For unsafe conditions, engage engineer early; saves time
- **Documentation**: Photograph site before/after; document all corrective work with timestamps
- **Permits first**: If work-without-permit SWO, file permit immediately; don't wait for inspection
- **Follow up**: Don't assume DOB will schedule inspection; follow up weekly
- **Keep copy of SWO**: Photograph/scan posted notice; shows exact cause and date
- **Notify all stakeholders**: General contractor, owner, lenders, insurance — SWO affects many parties

## Common Mistakes to Avoid

- Continuing work hoping SWO won't be enforced (criminal risk)
- Not filing corrective permit immediately (delays rescission)
- Submitting incomplete engineer certification (SWO not lifted)
- Ignoring SWO notice (problem doesn't disappear; gets worse)
- Not paying attached civil penalties (rescission blocked)
- Assuming partial work can continue without written DOB approval
