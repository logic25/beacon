# Place of Assembly (PA) Filing Guide — PACO

## Overview

**PA** stands for **Place of Assembly** — a specialized DOB filing for spaces used to gather people. Unlike ALT-1 or ALT-2 filings that permit construction, a PA filing results in a **PACO (Place of Assembly Certificate of Operation)**, certifying that a space is safe and legal for assembly purposes. PA is both a filing type AND a certification requirement.

> **Terminology Note:** This filing is commonly referred to as "PA" (not "PAA"). The resulting approval document is called a **PACO — Place of Assembly Certificate of Operation**. Older references may still use "PAA" but the current term is PA/PACO.

### What PA Is (And Is Not)

- **PA Filing:** Application to DOB declaring a space will be used for assembly and requesting plan review
- **PACO (Place of Assembly Certificate of Operation):** Formal approval document confirming space meets egress, safety, and occupancy requirements for assembly use
- **NOT the same as a CO:** A PACO does not replace a Building Certificate of Occupancy; it is a separate compliance document for assembly-specific requirements

---

## When PA Filing is Required

File PA for any space intended for assembly use if:

1. **Occupant load ≥ 75 persons** (regardless of food service), OR
2. **Assembly use that serves food or drink** (any occupant load)

### Common PA-Triggering Uses

- **Restaurants and bars** (food service)
- **Nightclubs and lounges** (assembly + alcohol)
- **Event spaces and banquet halls**
- **Theaters and cinemas**
- **Churches, temples, mosques, and houses of worship**
- **Gyms and fitness studios** (if occupant load ≥ 75)
- **Concert venues, performance spaces**
- **Conference centers and ballrooms**
- **Community centers, cultural spaces**
- **Wedding venues, reception halls**

### Example: Does This Need PA?

| Scenario | PA Required? | Reason |
|----------|---------------|--------|
| Restaurant, 120 seats | **Yes** | Food service + occupant load ≥ 75 |
| Yoga studio, 40 people capacity | **No** | Occupant load < 75; no food service |
| Office conference room, 80 people capacity, no food | **No** | Occupant load ≥ 75 BUT not assembly use (assembly ≠ office work) |
| Nightclub, 60 capacity | **Yes** | Food/beverage service (bar) triggers PA regardless of count |
| Church, 150 capacity | **Yes** | Occupant load ≥ 75 |
| Private event space, 30 people, catering permitted | **Maybe** | If catering is a primary use, PA may apply [VERIFY: Threshold for intermittent food service in small spaces?] |

## PA vs. ALT-1: When You Need Both

### ALT-1 + PA (Construction + Assembly Certification)
If converting a space to assembly use AND doing construction work:
- File **ALT-1** for the alteration/renovation work
- File **PA** for assembly certification

Example: Converting a commercial office to a restaurant → ALT-1 (construction) + PA (assembly use approval → PACO)

### PA Alone (Assembly in Existing Legal Space)
If a space already has a valid CO and you only want to change its use to assembly (without renovation):
- File **PA only**
- No construction or ALT-1 needed

Example: Existing retail space (valid CO, occupancy class = Mercantile) changing to restaurant → PA filing for PACO

## Occupancy Load Calculation

**Critical:** PA approval depends on accurate occupancy load. Load is calculated as:

**Occupant Load = Gross Floor Area ÷ Occupant Load Factor**

### Typical Occupant Load Factors (per NYC Building Code)

| Use | Load Factor (sq ft per person) |
|-----|--------------------------------|
| Restaurant/Dining | 15 sq ft/person |
| Bar/Lounge | 15 sq ft/person (for seated); 7 sq ft/person (standing room) |
| Nightclub/Dance Floor | 7 sq ft/person |
| Church/Auditorium (fixed seating) | 7 sq ft/person |
| Gym/Fitness (without specified seating) | 50 sq ft/person |
| Theater (fixed seating) | 7 sq ft/person |

[VERIFY: Are these factors current with the latest NYC Building Code? Any recent changes?]

**Conservative Approach:** When in doubt, use the more restrictive load factor. Understating occupant load can trigger DOB objections or late-stage requirements.

## Plan Examination and DOB Review

### Submission
File PA application with all supporting documents in [Build/BIS].

### DOB Examiner Assignment
Application assigned to an examiner from PA division. Review focuses on:
- Occupancy load calculation accuracy
- Egress compliance (exit widths, travel distance, path of travel)
- Emergency systems (lighting, signage)
- Life safety code compliance

### Common DOB Objections

| Objection | Reason | Solution |
|-----------|--------|----------|
| Exit width insufficient | Travel distance or layout requires wider exits | Increase door widths or add secondary exits |
| Occupant load too high | Egress cannot accommodate stated load | Reduce stated load; smaller seating/table arrangement |
| Travel distance exceeds code limit | Farthest seat > 250 ft from exit (varies by use) | Relocate exits, add intermediate doors, reduce occupant load |
| Emergency lighting inadequate | Lighting plan missing or insufficient coverage | Add emergency lighting per code; coordinate with FDNY |
| Exit signage missing or incorrect | Exits not clearly marked | Add/revise exit signs to be visible from all occupied areas |
| Kitchen hood/suppression not FDNY-approved | Missing FDNY permits for cooking operation | File Ansul/hood system with FDNY before PA approval |
| Sprinkler/fire alarm plans incomplete | Life safety systems not documented | Coordinate FDNY filings; submit system plans to DOB |

## PA (Public Assembly) Inspection

Once DOB approves the PA filing, a **Public Assembly (PA) Inspection** is scheduled. This is a live inspection of the actual space (not a plans review).

### What PA Inspectors Examine

- **Egress routes:** Verify actual doors, hallways, stairs match approved plans; test door operation
- **Exit signage:** Confirm illuminated exit signs are present, visible, correct height
- **Emergency lighting:** Test emergency lighting coverage throughout space; backup battery function
- **Seating arrangement:** Verify layout matches approved plan; table spacing; aisle widths
- **Fire extinguishers:** Present, accessible, properly mounted, current certification tags
- **Occupancy posting:** See occupant load posted at each entrance [VERIFY: Is this still required?]
- **Kitchen inspection** (if applicable): Hood, suppression, ventilation operational
- **FDNY systems:** Fire alarm, sprinkler, standpipe tested and operational (if present)

### PA Inspection Preparation Checklist

- [ ] Space ready for inspection (not mid-construction)
- [ ] Seating/layout finalized and matches plans
- [ ] All exit doors operational and unlocked (test locks per code)
- [ ] Emergency lighting installed and functional
- [ ] Exit signs illuminated and visible
- [ ] Fire extinguishers mounted and certified
- [ ] Occupancy load posted at main entrance(s)
- [ ] Kitchen hood and suppression system operational (if applicable)
- [ ] FDNY systems (fire alarm, sprinkler, standpipe) tested and operational
- [ ] Building Code-compliant furniture/fixtures in place

## FDNY Coordination: Fire Safety Systems

PA filings often trigger **FDNY filings** for fire protection systems. Coordinate with FDNY as part of PA process.

### Common FDNY Filings for Assembly Spaces

| System | When Required | Filing Type |
|--------|---------------|-------------|
| **Fire Alarm** | Buildings >5,000 sq ft or ≥100 occupants (varies) | FA-1 |
| **Sprinkler System** | Life hazard, occupancy type, building height dependent | FS-1 |
| **Standpipe** | Buildings >75 ft or >55,000 sq ft | ST-1 |
| **Hood/Ansul (Cooking)** | Kitchens with cooking operations | KF-1 |

[VERIFY: Current FDNY threshold requirements? Have these changed with recent code updates?]

### Coordination Timeline
1. PA filed with DOB
2. Simultaneously or shortly after, file FDNY plans (fire alarm, sprinkler, hood, etc.)
3. FDNY approves/objects to fire systems plans
4. Resolve any FDNY objections
5. FDNY issues permit(s)
6. Systems installed and inspected by FDNY
7. **After FDNY approval**, PA Inspection can proceed (inspector verifies FDNY systems operational)

**Key:** Do not schedule PA Inspection until FDNY approvals and inspections complete.

## Certificate of Operation: Approval and Renewal

### PA Approval
Once DOB approves PA filing and PA Inspection passes, **PACO (Certificate of Operation) is issued**. This formal document:
- Certifies space is legal and safe for assembly
- States authorized occupant load
- Lists any conditions or restrictions
- Must be posted prominently at entrance

## TPA (Temporary Place of Assembly): Events and Special Uses

TPA is a temporary PA — for one-time or limited-duration assembly events.

### When TPA is Used
- Wedding receptions in non-restaurant venues
- Corporate events in non-dedicated event spaces
- Temporary gatherings (food festival, fair)
- Pop-up retail/hospitality

## Restaurant/Bar PA Specifics

### Liquor License Coordination
If space is a restaurant, bar, or nightclub with alcohol service, coordinate **SLA (State Liquor Authority) licensing** with PA filing. Some jurisdictions require DOB PACO before SLA issues liquor license.

### Kitchen and Food Service
- Verify kitchen design meets Health Department + Building Code
- File FDNY hood and Ansul suppression systems
- Coordinate with NYC Department of Health for food service license (separate from DOB PA)

### Seating Arrangement
Restaurants/bars must show approved seating plan. Load is typically calculated at **15 sq ft per person** for seated dining, **7 sq ft per person** for standing room (bar areas).

## Common PA Pitfalls

| Pitfall | Impact | How to Avoid |
|---------|--------|-------------|
| Occupant load overstated | PA inspection fails; forced to reduce capacity or modify space | Conservative calculation; test egress flow |
| FDNY systems coordination delayed | PA inspection blocked until fire systems approved | Initiate FDNY filings concurrently with PA |
| Egress plan doesn't match reality | PA inspector finds mismatches; re-inspection required | Build out space to match plans before PA inspection |
| Exit signs missing or dark | PA inspection fails | Install and test all exit signage before inspection |
| Space used before Certificate issued | Violation issued; building sealed | Do not open to public until PA approved and PACO posted |
| TPA filed too late for event | Application rejected or delayed; event cancellation | File TPA at least [VERIFY: minimum advance notice?] days before event |
| Kitchen hood not FDNY-approved | Cooking use prohibited until FDNY approval | Coordinate hood/Ansul filing months in advance for restaurants |

## Food Service and Catering Coordination

If assembly space permits catering or food service:
- Verify zoning allows food service (some areas restrict)
- File FDNY hood/suppression for any cooking appliances
- Obtain NYC Health Department food service license (separate from DOB)
- Ensure kitchen meets health code (adequate handwashing, cold/hot storage, grease traps if required)

**Timeline Note:** Health Department licensing can take weeks; start early if catering is core to the use.
