# Letter of No Objection (LNO) Guide

## Overview

A Letter of No Objection (LNO) is a Department of Buildings document acknowledging that the Department has no objection to a property's current use, even though a Certificate of Occupancy (CO) is not on file or cannot be located. The LNO establishes legal acknowledgment of an existing lawful use, protecting the owner's ability to continue that use and facilitating future work on the building.

## What is an LNO and When is It Used?

### Definition

An LNO is a formal letter from the Department of Buildings stating:
- The Department has reviewed available records
- No objection exists to the property's current/proposed use
- The Department does not contest the lawful nature of the existing use
- The property may continue in this use or transition to another permitted use

### Key Distinction

An LNO is **NOT** a Certificate of Occupancy. It is an acknowledgment that a CO either:
- Was never issued (pre-1938 buildings or early conversions)
- Has been lost or misplaced
- Cannot be located in DOB records

A CO is proof of approval. An LNO is proof of non-objection to an existing condition.

### When to Pursue an LNO

LNOs are typically pursued in these scenarios:

1. **No CO on File**
   - Building predates modern CO requirements (pre-1938)
   - Building has been in continuous use for decades
   - Original CO cannot be located
   - Owner wants to formalize existing lawful use

2. **Existing Use Verification**
   - Building/space has been used for a particular purpose for many years
   - No record of original approval exists
   - Owner wants documentation for financing, insurance, or sale purposes
   - Space is legal non-conforming (grandfathered use)

3. **Transition Between Uses**
   - Owner wants to change use but cannot obtain CO for current use
   - LNO establishes current use is lawful/accepted
   - Supports application for new use or modification

4. **Pre-1938 Buildings**
   - Majority of NYC buildings built before modern CO system
   - Original CO often not available
   - Current use needs formalization for alteration permits or compliance
   - Owner planning renovations and needs clear use documentation

5. **Financing or Sale**
   - Lender requires proof of lawful use before mortgage refinance
   - Buyer's attorney requires use verification before purchase
   - Insurance company needs documentation of approved use
   - LNO provides official DOB acknowledgment

6. **Missing COs from Building Alterations**
   - Building has undergone multiple alterations
   - CO for alteration work cannot be located
   - Previous owner may have lost documentation
   - Current use was approved by prior CO now missing

**Validity:**

### Type 2: Informal LNO or Waiver of Objection [VERIFY if DOB currently issues]

**What it is:**
- Less formal acknowledgment obtained during permit application process
- DOB's acceptance of use for purposes of issuing a work permit
- Not a standalone letter but notation in permit file
- May not be suitable for financing purposes without formal LNO

**When obtained:**
- During application for renovation or alteration work permit
- Owner proposes work in building with no CO on file
- DOB accepts the use as lawful for permit purposes

**Differences from Formal LNO:**
- Not issued as separate formal letter
- May only exist as file notation
- Less suitable for independent financing/sale documentation
- Easier to obtain but less official

**When to Pursue This:**
- Owner needs use approval for construction permits only
- Not pursuing sale or refinancing
- Timeline is critical (faster than formal LNO)
- Cost-sensitive project

## When to Pursue: LNO vs. CO vs. Letter of Completion

### Decision Tree

| Situation | Best Option | Why |
|-----------|------------|-----|
| **Building has valid CO on file** | Use existing CO | Most direct; no action needed |
| **Recent alteration with valid Completion Notice** | Letter of Completion | CO was issued; just need formal completion |
| **Building pre-1938, no CO ever issued, no current work planned** | Formal LNO | Formalizes existing lawful use |
| **Owner wants to verify/formalize current use for financing** | Formal LNO | Establishes DOB acknowledgment of use |
| **Owner needs use approval quickly for construction permit only** | Informal waiver (if available) | Faster; embedded in permit process |
| **Building has CO but recent work has no Completion Notice** | Letter of Completion application | Completes the record for final work |
| **Use is non-conforming (grandfathered), no CO, owner needs documentation** | Formal LNO | Only way to formalize grandfathered use |

### Scenarios and Recommendations

**Scenario A: Pre-1938 Tenement with No CO, Planning Renovation**
- **Recommendation:** File formal LNO **before** filing renovation/alteration permits
- **Reason:** Having LNO streamlines subsequent permit approvals; DOB already confirmed use is acceptable
- **Timeline:** File LNO (4-8 weeks) then file alteration permit (timeline separate)

**Scenario B: 1970s Office Building, Need Refinance, No CO on File**
- **Recommendation:** File formal LNO immediately
- **Reason:** Lender will almost certainly require official DOB documentation; informal acknowledgment insufficient
- **Timeline:** Budget 6-10 weeks for LNO; do this first before approaching lender

**Scenario C: Ground Floor Restaurant, Want to Change Use to Retail, No CO**
- **Recommendation:** File formal LNO for restaurant use first, then apply for use-change permit
- **Reason:** Establishes existing restaurant use is lawful; supports approval for different use in same space
- **Timeline:** LNO (4-8 weeks) + Use-change permit (2-4 weeks additional)

**Scenario D: Building with CO for Original Use, Building Owner Wants to Verify Current Mixed Use (after informal alterations)**
- **Recommendation:** Formal LNO for current use may be appropriate
- **Reason:** Original CO may not reflect current actual conditions; LNO updates record
- **Alternative:** Letter of Compliance from licensed professional [VERIFY if available]

**Scenario E: Owner Needs Quick Use Approval for Emergency Permit or Minor Work**
- **Recommendation:** Informal waiver/acknowledgment during permit application (if available)
- **Reason:** Speed is critical; formal LNO can be pursued later if needed
- **Timeline:** Can be embedded in permit application (typically 1-2 weeks)

## Filing Process for Formal LNO

### Phase 1: Pre-Application Research and Preparation

#### Step 1: Research Existing DOB Records

1. **Obtain Building's DOB Record**
   - Title Search / Building History Report from licensed record searcher
   - Or: Visit DOB in person at Manhattan office to review records
   - Or: Request records via FOIL (Freedom of Information Law) request [VERIFY current process]

2. **Document Review**
   - Identify most recent CO on file (if any)
   - Note date of issuance and permitted use
   - Identify major alterations with permits/COs
   - Document gaps (where permits/alterations lack recorded completion)
   - Note any violations still open on property [VERIFY if this impacts LNO eligibility]

3. **Create Timeline Document**
   - Establish when building was constructed (from property records)
   - List all known permits, COs, and alterations
   - Identify period when building has been in continuous use
   - Note any documented changes of use
   - This document supports your LNO application narrative

#### Step 2: Gather Supporting Documentation

4. **Current Use Verification**
   - **Lease or Occupancy Documentation:** Current lease or occupancy agreement if applicable [VERIFY if required]
   - **Tax Records:** City property tax records showing current use classification
   - **Business Licenses:** Current NYC business license, food service license, or industry-specific license (if applicable)
   - **Insurance Records:** Current insurance policy showing building use/occupancy
   - **Photographs:** Current interior/exterior photographs showing the use (office furniture, retail fixtures, signage, etc.)
   - **Utility Bills:** Copies of recent bills for the address showing occupancy

5. **Historical Documentation** (if pre-1938 or long-standing use)
   - **Old Leases or Documents:** If available, documentation of use 10, 20, or 30 years ago
   - **Previous Owner Information:** Contact info or documentation about prior ownership and use history
   - **Google Street View / Historic Photos:** Screenshots showing building use over years
   - **Newspaper Records:** Any historical records documenting building's use
   - **Testimonial Letters:** Statements from long-time tenants or community members about continuous use [VERIFY if accepted]

6. **Professional Documents**
   - **Floor Plan:** Current or recent floor plans showing layout and use (even if informal sketch if professional plan not available)
   - **Professional Certification:** Letter from licensed architect or engineer certifying current use and physical conditions [VERIFY if required]
   - **Compliance Inspection Report:** Licensed professional's report confirming building meets standards for the stated use [VERIFY if required]

#### Step 3: Determine Applicable Building Code and Zoning

7. **Verify Zoning Compliance**
   - Check NYC Department of City Planning Zoning Map
   - Confirm proposed/current use is permitted in zoning district or is grandfathered
   - Obtain zoning summary from Zoning Information System (ZIS) [VERIFY if included in LNO application process]
   - Document if use is grandfathered under prior zoning or code [VERIFY process]

8. **Identify Building Code Requirements**
   - Determine if building must meet current Building Code for stated use
   - Or if pre-1938 building has grandfathered status
   - Document applicable code (NYC Building Code 2023 or prior)
   - Note any variance or exemptions that apply [VERIFY]

### Phase 2: Application Preparation

#### Step 4: Complete LNO Application

9. **Obtain and Complete Application Form**
   - **Form:** NYC Department of Buildings LNO application form [VERIFY current form number - may be combined with other applications]
   - **Sections to Complete:**
     - Property information (address, block/lot, owner name)
     - Current use description (in detail)
     - Building square footage and occupancy
     - Number of units/tenants if applicable
     - Requested effective date
   - **Sign and Date:** Owner or authorized representative signature required [VERIFY]

10. **Prepare Narrative Letter**
    - **Content:** Explain the property's history and current use
    - **Key Points to Address:**
      - When building was constructed
      - What use has been continuous (if applicable)
      - Why CO is not on file (pre-1938, lost, etc.)
      - Current occupancy status
      - Any code compliance measures in place
      - Why use is lawful (grandfathered if applicable)
    - **Tone:** Professional, factual, clear
    - **Length:** 1-2 pages typical
    - **Professional Signature:** If required, architect/engineer certification

11. **Prepare Floor Plan(s)**
    - **Existing Conditions Plan:** Show layout with uses labeled
    - **Scale:** Drawn to scale if professionally prepared; hand-drawn sketch acceptable if clearly labeled [VERIFY]
    - **Detail:** Show walls, doors, major fixtures, egress paths
    - **Occupancy:** Indicate occupancy loads if applicable (e.g., 50 office workers, 30 retail customers)
    - **Professional Stamp:** If prepared by licensed professional [VERIFY if required]

12. **Create Documentation Package**
    - **Transmittal Letter:** Cover letter listing all documents
    - **Organize in Order:**
      - Application form (original with signature)
      - Narrative letter
      - Floor plans
      - Supporting documents (leases, licenses, photos, etc.)
      - Proof of ownership or authorization to request LNO [VERIFY what's required]
    - **Copies:** Prepare 2-3 complete copies (one original)
    - **Binding:** Use brads or folders (not stapled) [VERIFY current preference]

### Phase 3: Submission

#### Step 13: Submit to DOB

13. **Identify Correct Department**
    - **Manhattan:** DOB - 33 Beaver Street (Main office)
    - **Brooklyn, Queens, Bronx, Staten Island:** Borough DOB offices [VERIFY current addresses]
    - **One-Stop Shop:** Consider submitting at appropriate borough One-Stop Shop [VERIFY if LNO submissions accepted there]

14. **Submit Application**
    - **In Person:** Deliver to DOB in-person for official date stamping
    - **By Mail:** Send via certified mail with return receipt [VERIFY if email submissions available]
    - **Obtain Receipt:** Get submission receipt with application number and date
    - **Retain Copy:** Keep copy of received application for your records

15. **Document Submission**
    - **Application Number:** Obtain and record official DOB application number
    - **Date of Submission:** Note official receipt date
    - **Assigned Borough/Office:** Confirm which office will process
    - **Contact Person:** Attempt to get name of assigned reviewer (may not be available initially)

### Phase 4: DOB Review and Issuance

#### Step 16: DOB Records Review

16. **Initial Review (1-2 weeks)**
    - DOB checks application for completeness
    - DOB reviews building file records
    - Searches for any prior permits, COs, violations
    - Documents timeline of property's history

17. **Completeness Check**
    - If application incomplete, DOB will issue deficiency notice
    - Deficiency letter specifies missing documents or information
    - Applicant has [VERIFY duration] to respond with additional information
    - Multiple deficiency rounds may occur

#### Step 17: Use Verification and Acceptance

18. **DOB Analysis of Use**
    - DOB evaluates whether stated use is lawful and non-conflicting with records
    - Checks if use is grandfathered or permitted in current zoning
    - Verifies use doesn't create code violations
    - May contact property owner for clarification [VERIFY]

19. **No Objection Determination**
    - If no conflicts found, DOB prepares "No Objection" letter
    - Letter states DOB has no objection to the use
    - Letter documents what use was reviewed and approved for
    - Letter is signed by authorized DOB official

#### Step 18: LNO Issuance

20. **Receipt of LNO Letter**
    - Official letter from NYC Department of Buildings
    - On DOB letterhead with official seal [VERIFY]
    - Includes:
      - Property address and block/lot
      - Description of use for which no objection exists
      - Date of issuance
      - Official signature and title of DOB official
      - Application number and reference number
    - Issued in original (usually only one original) and copies [VERIFY]

21. **LNO Validity and Use**
    - **Validity:** No expiration date (valid indefinitely unless use changes)
    - **Use:** Provide to lenders, insurers, buyers, or for permit applications
    - **Reproduction:** Can be photocopied for official submissions
    - **Confidence Level:** Highest official acknowledgment of existing use without full CO

## Research Requirements: Digging into Building Records

### Where to Find Building Records

**1. DOB in Person Records Review**
- Location: 33 Beaver Street (Manhattan) or borough offices
- Access: Public access to building files, but some documents restricted [VERIFY]
- What's Available: Permits, violation history, CO records (if on file)
- Time Required: 1-3 hours typically
- Cost: Free

**2. Licensed Record Searcher**
- Services: Complete building history report with all permits, COs, violations
- Cost: $200-500 typical [VERIFY]
- Turnaround: 1-2 weeks
- Advantage: Professional organization, expert analysis
- Companies: Searching Title, Building Records, or similar NYC firms [VERIFY current firms]

**3. FOIL (Freedom of Information Law) Request**
- What: Request all building records from DOB via formal FOIL process
- Cost: Typically free or minimal copying fee
- Turnaround: 20 business days (can be extended) [VERIFY]
- Advantage: Gets all documents DOB has
- Disadvantage: Slow; documents may be disorganized

**4. NYC Property Tax Records (ACRIS)**
- Where: Assessments and City Record Information System (ACRIS)
- Access: Online via City records website
- What's Available: Deed history, tax assessment, use classifications over time
- Cost: Free
- Advantage: Shows historical use classifications

**5. NYC Department of City Planning**
- Access: Online Zoning Information System (ZIS)
- What's Available: Current and prior zoning, overlays, restrictions
- Cost: Free
- Useful for: Verifying grandfathered status if use pre-dates current zoning

**6. NYC Parks Department**
- If applicable: Building near or under park; Parks may have records of subsurface rights or restrictions [VERIFY]

### What to Look For in Records

**Priority Documents:**
- Original building CO (if exists)
- Most recent permit or alteration work
- Most recent Completion Notice or Certificate of Occupancy
- Violation history (what violations exist or were resolved)
- Zoning change documentation (if any)

**Timeline Indicators:**
- Building construction date
- Date of use change (if ever occurred)
- Dates of major permits/alterations
- Dates of violations and resolutions

**Gaps in Records:**
- Years with no documented permits despite ongoing alterations
- Missing COs for known work
- Pre-1938 buildings (almost never have original CO on file)

## Common Scenarios and Approaches

### Scenario 1: Pre-1938 Building (Tenement, Loft, Historic Building)

**Challenge:** Building built before modern Building Code and CO system; original CO never issued or never located

**Characteristics:**
- Building over 85 years old
- Current use (residential, light manufacturing, office) has been continuous
- No CO on file in DOB records
- Owner has good faith use for decades or inherited building in existing condition

**LNO Approach:**
1. Document building construction date and original use from historical records
2. Gather evidence of continuous occupancy (tax records, insurance, prior leases if available)
3. If use has changed, document when and that it's been stable
4. Include photographs and description of current physical condition
5. Have licensed professional (architect) certify current occupancy and code compliance
6. In narrative, explain pre-1938 status and why CO not on file
7. Submit with supporting documentation

**Timeline:** 4-8 weeks
**Success Rate:** High - DOB generally accepts pre-1938 buildings with evidence of continuous lawful use

### Scenario 2: Missing CO After Alteration Work (1970s-1990s)

**Challenge:** Building had alteration work permitted and completed, but CO (or Completion Notice) cannot be located

**Characteristics:**
- Building has prior CO on file (pre-alteration)
- Work performed per permit (permit on file in DOB)
- Completion notice/new CO should have been issued but cannot be found
- Prior owner may have lost documentation
- Current owner purchased with original CO, unaware of alteration CO gap

**LNO Approach:**
1. Locate original CO in DOB records
2. Locate permit for alteration work
3. Search for Completion Notice or post-alteration CO (may not exist in file)
4. Contact DOB to determine if Completion Notice was actually issued [VERIFY process]
5. If not issued, may need to file:
   - Letter of Completion application (to finalize prior work), OR
   - LNO confirming use post-alteration
6. Narrative should explain gap and document that work was lawfully permitted

**Timeline:** 2-6 weeks (depends on DOB's records availability)
**Success Rate:** High - Clear permit history supports approval

**Alternative:** Instead of LNO, consider filing Letter of Completion to officially close out the prior alteration permit

### Scenario 3: Grandfathered Non-Conforming Use

**Challenge:** Building's current use is not permitted under current zoning or code, but use pre-dates current regulations (grandfathered)

**Characteristics:**
- Building is in district where current use is not permitted (e.g., residential in commercial zone, manufacturing in residential zone)
- Use has been continuous since before zoning change
- Original zoning permit/CO may exist but be old
- No current CO because use is no longer permitted in district

**LNO Approach:**
1. Document when building use began (from historical records)
2. Document when zoning changed to make use non-conforming
3. Establish continuous use since pre-zoning-change date
4. Obtain attorney letter or zoning analysis confirming grandfathered status [VERIFY if required]
5. Gather evidence of continuous uninterrupted use (leases, tax records, business history)
6. In narrative, specifically address grandfathered status and why use remains legal
7. Include legal analysis if building is in complex zoning situation

**Timeline:** 6-10 weeks (longer due to research and potential legal review)
**Success Rate:** Good if grandfathered status can be clearly established; may require legal expertise

### Scenario 4: Recent Mixed-Use Development (no CO for some portions)

**Challenge:** Building underwent major renovation creating multiple uses, but only partial COs issued

**Characteristics:**
- Building has undergone recent significant renovation (last 5-10 years)
- Some portions have valid COs (e.g., commercial ground floor)
- Other portions may lack COs or have incomplete permits (e.g., residential units or office spaces)
- Current owner wants to finalize records or sell/refinance

**LNO Approach:**
1. Obtain all permits and COs on file for renovation work
2. Document which uses have valid COs and which do not
3. Gather current documentation for uses lacking COs (lease for office space, residential occupancy proof, etc.)
4. Determine if incomplete COs can be finalized or if LNO is better path
5. If LNO path: Submit separately for each use/space that lacks CO
6. Include current floor plans showing all uses
7. Include proof of current lawful occupancy and code compliance

**Timeline:** 6-12 weeks (may involve multiple LNO applications or completion filings)
**Success Rate:** Variable - depends on whether prior permits exist and can be completed vs. new uses needing establishment

## Relationship to Certificate of Occupancy and Building Permits

### When LNO is a Gateway to Other Permits

**Scenario A: Owner Plans Renovation**
1. Building has no CO on file
2. File LNO for current use → obtain LNO letter
3. With LNO on file, file renovation/alteration permit application
4. During alteration work, obtain new CO at completion
5. **Result:** Building now has proper CO for current use

**Scenario B: Owner Plans to Change Use**
1. Building has no CO for current use
2. File LNO for **current** use
3. Once LNO approved, can apply for **use-change permit**
4. Upon permit completion, obtain CO for **new** use
5. **Result:** Old use formally acknowledged (LNO); new use formally approved (CO)

**Scenario C: Owner Needs Certificate of Occupancy**
1. LNO is not a CO; it only acknowledges no objection
2. If owner truly needs CO (lender requires it for new use), must pursue alteration permit + new CO
3. LNO can be stepping stone: proves use is acceptable to DOB → easier path to permit approval
4. But ultimately, CO requires permitted work and final inspection

## Common Mistakes to Avoid

### Mistake 1: Submitting Incomplete Documentation
**Problem:** Missing proof of current occupancy, unclear floor plans, insufficient narrative
**Impact:** DOB issues deficiency; adds 2-4 weeks
**Prevention:** Use checklist above; ensure every required document present; have professional review before submission

### Mistake 2: Not Researching Prior COs Thoroughly
**Problem:** Applicant thinks building has no CO, but one is on file
**Impact:** DOB rejects LNO as unnecessary or confusing
**Prevention:** Hire licensed record searcher or spend time at DOB reviewing files before applying

### Mistake 3: Pursuing LNO When Other Remedy Exists
**Problem:** Owner files LNO but could have easily obtained Letter of Completion for prior work
**Impact:** Unnecessary time and cost
**Prevention:** Confirm no CO exists for current condition and no completion notice path available

### Mistake 4: Insufficient Explanation of Grandfathered Status
**Problem:** Owner submits LNO for use that doesn't appear to be permitted in current zoning, without legal analysis
**Impact:** DOB questions whether use is actually grandfathered; may deny or delay approval
**Prevention:** If grandfathered, include zoning analysis or attorney letter confirming status

### Mistake 5: Misrepresenting Current Use or Conditions
**Problem:** Floor plans or narrative inaccurate about actual space layout or occupancy
**Impact:** DOB discovers discrepancy during inspection or review; LNO can be revoked
**Prevention:** Ensure all documentation accurately reflects current conditions; professional verification helps

### Mistake 6: Treating LNO as Equivalent to CO
**Problem:** Owner uses LNO as if it permits new uses or construction
**Impact:** Confusion with future permit applications; may delay approvals
**Prevention:** Understand LNO is acknowledgment of existing use only; doesn't authorize new work or use changes

### Mistake 7: Allowing Conditions to Change After LNO Approval
**Problem:** LNO approved for "office use"; building then converts to residential
**Impact:** LNO becomes outdated or potentially invalid [VERIFY]
**Prevention:** If use actually changes, file new LNO; don't rely on old LNO for new use

## When to Seek Professional Help

### Consult a Licensed Architect/Professional

- Building is pre-1938 or has complex history
- Grandfathered status is unclear or legally complex
- Use is unusual or non-standard
- Code compliance questions exist
- DOB may require professional certification
- Building has prior violations or issues

### Consult a Real Estate Attorney

- Grandfathered use requires legal analysis
- LNO is for purchase/sale transaction (buyer protection)
- Complex zoning situation or overlay districts
- Building in historic district with special requirements
- Prior legal disputes over building use

### Consult an NYC Expediter

- Navigating DOB submissions and follow-ups
- Concurrent with other permit applications
- Unclear timeline or next steps
- Previous denied applications
- Complex project with multiple Department coordination

## Key Takeaways

1. **LNO ≠ CO** - LNO is acknowledgment of no objection; CO is permit approval
2. **Pre-1938 buildings likely to need LNO** - Most don't have original COs on file
3. **Best timing:** File LNO **before** planning major work or sale/refinance
4. **Document thoroughly** - Current photos, leases, tax records, and business licenses critical
5. **Research building history** - Helps explain why CO missing and establishes lawful use timeline
6. **Professional help valuable** - Especially for grandfathered uses or complex histories
7. **Timeline is 4-8 weeks minimum** - Budget adequate time; don't rush
8. **Cost is manageable** - $1,000-3,000 typical for complete application
9. **Success rate high** - Most straightforward cases approved if documented properly
10. **LNO lasts indefinitely** - Valid forever unless use actually changes
