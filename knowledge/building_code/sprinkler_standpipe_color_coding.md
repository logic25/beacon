# Sprinkler and Standpipe Painting Color Coding

**Source:** New York Fire Sprinkler Contractor's Association, Inc.

Color coding requirements vary based on system type and whether the work is new construction, alteration, or existing.

## Color Code Key

- **RED** = Standpipe system components
- **GREEN** = Dedicated sprinkler system components
- **PAINT COLOR** (either red or green as applicable) = As specified below

---

## New Construction (as of March 2, 2010)

### Paint RED:
- Stand alone sprinkler cross connections and risers
- All portions of standpipe system including valve handles
- Combination systems risers and cross connections

### Paint GREEN:
- All handles of valves serving dedicated sprinklers
- Handles of valves serving combination systems

---

## Alterations (After March 2, 2010)

### Paint RED:
- Existing handles of valves serving standpipe systems and existing standpipe risers that are exposed during alterations
- Cross connections and risers for stand alone sprinkler systems that are exposed during alterations

### Paint GREEN:
- All handles of valves serving dedicated sprinklers that are exposed during alterations
- All handles of valves serving combination systems

---

## Existing Buildings (As of June 2, 2010)

### Paint RED:
- Exposed portions of stand alone sprinkler system risers and cross connections
- All portions of exposed standpipe systems
- Exposed valve handles serving the standpipe system

### Paint GREEN:
- All handles of valves serving sprinkler system
- All handles of valves serving combination systems

---

## Under Construction (Work not signed off by June 2, 2010 and started prior to March 2, 2010)

### Paint RED:
- All exposed portions of the standpipe system
- Exposed handles of valves serving the standpipe system
- Sprinkler risers and cross connections that are also used for standpipe system

### Paint GREEN:
- Handles of valves serving combination systems
- All handles of valves serving sprinkler system

---

## Quick Reference

| Component | Color |
|---|---|
| Standpipe risers and system | RED |
| Sprinkler-only valve handles | GREEN |
| Combination system valve handles | GREEN |
| Stand alone sprinkler cross connections/risers (new construction) | RED |
| Dedicated sprinkler valve handles | GREEN |

## Practical Notes

- Common inspection objection: wrong color coding on sprinkler/standpipe valve handles
- New PMs should verify color coding during site visits before sign-off inspections
- Combination systems are the most common source of confusion — valve handles are GREEN even though the system serves both sprinkler and standpipe
